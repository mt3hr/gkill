Lebensprotokoll-Ansicht

Bildschirm zum Durchsuchen von Lebensprotokollen mit Miniaturansichten. Unterstuetzt mehrere Anzeigemodi und Filterfunktionen.

Symbolleistenschaltflaechen

Schaltflaeche Beschreibung
☰ Seitenleiste Oeffnet/schliesst die Seitenleiste zum Filtern und Navigieren.
📄 Details Zeigt/verbirgt das Detailpanel des Datensatzes.
📊 Zusammenfassungsansicht Zeigt den Bildschirm fuer statistische Analyse an.
📅 Kalender Zeigt die Kalenderansicht an.
🗺 Karte Zeigt Datensaetze mit GPS-Informationen auf der Karte an.
⚙ Einstellungen Oeffnet den Einstellungsbildschirm der Anwendung.
❔ HilfeZeigt die Hilfe fuer den aktuellen Bildschirm an.

Hauptfunktionen

  • Mehrspaltige Miniaturansichten: Miniaturansichten koennen in 1 bis 10 Spalten angezeigt werden.
  • Filterfunktion: Filtert Datensaetze nach Typ, Tag, Datumsbereich, Ablage usw. ueber die Seitenleiste.
  • Detailpanel: Bei Auswahl eines Datensatzes werden detaillierte Informationen angezeigt.
  • Statistische Analyse: Zeigt statistische Daten und Trends der Datensaetze an.
  • Kartenanzeige: Zeigt Datensaetze mit Standortinformationen auf der Karte an.
Tipp: Sie koennen die Anzahl der Miniaturansichtsspalten (1-10) in den Einstellungen anpassen, um verschiedene Bildschirmgroessen zu unterstuetzen.
Tipp: Der Standardwert des Anzeigezeitraums kann in den Einstellungen unter „Lebenslog-Ansicht Anzeigetage" geändert werden.

FAB (Floating-Action-Schaltflaeche) "+"

Durch Klicken auf die "+"-Schaltflaeche oeffnet sich ein Menue zum schnellen Erstellen von Datensaetzen:

NumerischZeichnet numerische Daten auf.
LesezeichenSpeichert ein URL-Lesezeichen.

Tastenkürzel: Drücken Sie Ctrl+V, um den Inhalt der Zwischenablage direkt als Datei zu speichern. Nützlich zum schnellen Speichern von Screenshots oder kopierten Bildern.

Tags verwenden

Ein Tag hinzufugen

  1. Halten Sie einen Datensatz gedruckt oder klicken Sie mit der rechten Maustaste, um das Kontextmenu zu offnen.
  2. Wahlen Sie "Tag hinzufugen".
  3. Geben Sie den Tag-Namen ein (vorhandene Tags werden als Vorschlage angezeigt).

Ein Tag bearbeiten oder loschen

  1. Halten Sie das zu andernde Tag gedruckt oder klicken Sie mit der rechten Maustaste darauf.
  2. Wahlen Sie "Bearbeiten" oder "Loschen" aus dem Menu.

Tag-Strukturverwaltung

Sie konnen Anzeigenamen und Reihenfolge der Tags unter Einstellungen > Tag-Struktur verwalten.

Tipp: Wenn Sie #Tag-Name in der Notizblock-Syntax schreiben, wird der Datensatz automatisch mit einem Tag versehen. Weitere Details finden Sie in der Notizblock-Syntaxreferenz.

Datei-Upload-Details

  1. Tippen Sie auf die "+"-Schaltflache und wahlen Sie "Datei hochladen".
  2. Wahlen Sie das Ziel-Repository (Repository fur die Dateiverwaltung).
  3. Bestatigen und passen Sie das zugehorige Datum und die Uhrzeit an.
  4. Wahlen Sie die hochzuladende(n) Datei(en) aus (mehrere Dateien moglich).

Unterstutzte Formate: Bilder (JPEG/PNG/GIF/WEBP), Videos (MP4 usw.), PDF, ZIP und andere.

ZIP-Dateien konnen in ihrem Inhalt durchsucht werden (Rechtsklick > ZIP-Inhalt offnen).

Groessenbeschrankung: Gemas Serverkonfiguration (-1 fur unbegrenzt).

Tipp: Drucken Sie Ctrl+V, um den Inhalt der Zwischenablage direkt als Datei zu speichern.

GPS-Karte verwenden

Eine GPS-Aufzeichnung hochladen

  1. Tippen Sie auf die "+"-Schaltflache und wahlen Sie "Datei hochladen".
  2. Wahlen Sie das fur GPS-Aufzeichnungen vorgesehene Repository.
  3. Wahlen Sie die GPX-Datei aus.
  4. Aktivieren Sie die Kartenschaltflache, um die Anzeige zu bestatigen.

Google Maps API-Schlussel konfigurieren

Gehen Sie zu Einstellungen > Anzeigeeinstellungen > Google Maps API-Schlussel und geben Sie Ihren Schlussel ein.

Tipp: Holen Sie sich Ihren API-Schlussel in der Google Cloud Console, indem Sie die Maps JavaScript API aktivieren.
