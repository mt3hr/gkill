KI-Integration

Durch die Verbindung von KI-Assistenten wie Claude mit gkill können Sie Ihre Lebensprotokoll-Einträge per natürlicher Konversation suchen, hinzufügen und bearbeiten.

Was Sie tun können

AktionBeispiel
Einträge suchen„Zeige mir meine Mittagessen-Notizen der letzten Woche" / „Fasse diesen Monat Ausgaben zusammen"
Einträge hinzufügen„Füge eine Notiz für heute hinzu" / „Erstelle eine Aufgabe"
Einträge aktualisieren / löschen„Markiere diese Aufgabe als erledigt"
GPS / Dateien abrufen„Zeige meine heutige Route"
Einstellungen prüfen„Liste meine Vorlagen auf"

Integrationstypen

TypBeschreibung
Nur-LesenDie KI kann Einträge nur anzeigen und suchen. Hinzufügen oder Ändern ist nicht möglich.
Nur-SchreibenDie KI kann Einträge nur hinzufügen, aktualisieren und löschen.
Lesen & SchreibenDie KI kann Einträge anzeigen, hinzufügen, aktualisieren und löschen.

Einrichtung (Claude Desktop)

Fügen Sie Folgendes zur Claude-Desktop-Konfigurationsdatei (claude_desktop_config.json) hinzu.

Nur-Lesen-Integration

{
  "mcpServers": {
    "gkill": {
      "command": "npx",
      "args": ["-y", "gkill-mcp-server"],
      "env": {
        "GKILL_BASE_URL": "http://localhost:9999",
        "GKILL_USER": "ihre-benutzer-id",
        "GKILL_PASSWORD_SHA256": "sha256-hash-ihres-passworts"
      }
    }
  }
}
Tipp: GKILL_PASSWORD_SHA256 ist der SHA256-Hash Ihres Passworts. Sie können ihn mit den auf Ihrem Betriebssystem verfügbaren Tools generieren.

Einrichtung (Claude Code)

npx @anthropic-ai/claude-code mcp add gkill \
  --command npx \
  --args "-y,gkill-mcp-server" \
  --env GKILL_BASE_URL=http://localhost:9999 \
  --env GKILL_USER=ihre-benutzer-id \
  --env GKILL_PASSWORD_SHA256=ihr-hash

Fernzugriff

Machen Sie Ihren gkill-Server über das Internet zugänglich und setzen Sie GKILL_BASE_URL auf seine öffentliche Adresse. Aus Sicherheitsgründen wird die Aktivierung von TLS (HTTPS) dringend empfohlen.

Tipp: TLS-Einstellungen können unter Servereinstellungen konfiguriert werden.
