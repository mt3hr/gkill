Erste Schritte

gkill ist eine Anwendung zum Aufzeichnen und Durchsuchen von taglichen Lebenslogs.

Was Sie mit gkill tun konnen

Aufzeichnungstyp Beschreibung
Notiz Zeichnet Textnotizen auf
Zeitstempel Zeichnet Start- und Endzeiten auf
Aufgabe Verwaltet Aufgaben auf einem Kanban-Board
Stimmung Zeichnet die Stimmung als Wert von 0 bis 10 auf
Ausgabe Zeichnet Ausgabenbetrage und Details auf
Lesezeichen Speichert URLs von Webseiten
Numerische Aufzeichnung Zeichnet beliebige numerische Daten auf
Datei Ladt Bilder oder Dokumente hoch und zeichnet sie auf

Ersteinrichtung

  1. Starten Sie gkill und rufen Sie http://localhost:9999 in Ihrem Browser auf.
  2. Melden Sie sich mit der Benutzer-ID admin und ohne Passwort an.
  3. Nach der Anmeldung wird empfohlen, im Einstellungsbildschirm ein Passwort festzulegen.
  4. Konfigurieren Sie im Einstellungsbildschirm grundlegende Einstellungen wie Sprache und dunkles Design.

Erstellen Sie Ihre erste Aufzeichnung

  1. Nach der Anmeldung wird der in den Einstellungen festgelegte Standardbildschirm angezeigt.
  2. Tippen Sie auf die "+"-Schaltflache unten rechts auf dem Bildschirm.
  3. Wahlen Sie "Notizblock".
  4. Geben Sie im Textbereich ein, was Sie aufzeichnen mochten (z. B.: Heute gab es Curry zum Mittagessen).
  5. Drucken Sie die Schaltflache "Speichern", um die Aufzeichnung zu erstellen.
Tipp: Durch langes Drucken der "+"-Schaltflache wird der Notizblock direkt geoffnet.

Glossar

Begriff Beschreibung
Datensatz Allgemeiner Begriff fur Aufzeichnungen in gkill. Umfasst alle Aufzeichnungstypen einschliesslich Notizen, Zeitstempel, Aufgaben und mehr
Notizblock-Syntax Texteingabesyntax, die im Notizblock verwendet wird. Durch Hinzufugen von Symbolen am Anfang jeder Zeile konnen verschiedene Arten von Aufzeichnungen gleichzeitig erstellt werden
Zeitstempel Zeiterfassungsfunktion. Zeichnet die Start- und Endzeiten von Arbeiten oder Aktivitaten auf
Saihate Bildschirm zum sicheren Hinzufügen von Aufzeichnungen auch vom Sperrbildschirm aus. Dient als Hub zum Hinzufügen von Aufzeichnungen
Notizblock-Syntax Bezeichnet das textbasierte Format zur Eingabe von Aufzeichnungen
Lebenslog-Ansicht Bildschirm zum Auflisten und Suchen von Aufzeichnungen
Aggregationsansicht Funktion zur Anzeige von Datenaggregationen und Statistiken
Verwandte Informationen Funktion, die automatisch nach verwandten Informationen sucht und diese anzeigt
Repository Speicherort fur Aufzeichnungen. Sie konnen mehrere Repositories fur verschiedene Zwecke verwenden
Profil Eine Sammlung von Einstellungen, die nach Gerat gruppiert sind
