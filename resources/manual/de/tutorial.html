Tutorial

Hier wird die Verwendung von gkill in einem Grundlagen- und einem Fortgeschrittenenteil erklärt. Wenn Sie gkill zum ersten Mal verwenden, lesen Sie bitte zunächst den Grundlagenteil.

Grundlagen

1. Bei gkill anmelden

Erster Start

Wenn Sie gkill zum ersten Mal starten, wird automatisch die Kontoregistrierungsseite angezeigt.

  1. Geben Sie eine Benutzer-ID ein (dies wird Ihr Kontoname).
  2. Geben Sie ein Passwort ein und bestätigen Sie es durch erneute Eingabe.
  3. Legen Sie ein Administratorpasswort fest (wird für die Serververwaltung verwendet).
  4. Drücken Sie die Schaltfläche „Registrieren“, um das Konto zu erstellen.
  5. Sie werden zur Anmeldeseite weitergeleitet. Melden Sie sich mit dem erstellten Konto an.

Normale Anmeldung

  1. Geben Sie Ihre Benutzer-ID in das Eingabefeld für die Benutzer-ID ein.
  2. Geben Sie Ihr Passwort in das Passwortfeld ein.
  3. Klicken Sie auf die Schaltfläche „Anmelden“ oder drücken Sie die Eingabetaste.
Tipp: Die Sitzung nach der Anmeldung ist 30 Tage gültig. Bis zum Ablauf ist keine erneute Anmeldung erforderlich.

2. Einträge hinzufügen

In gkill können Sie verschiedene Arten von Lebensprotokollen aufzeichnen. Fügen Sie Einträge über die Schaltfläche „+“ unten rechts auf dem Bildschirm hinzu.

EintragstypVerwendungEingabebeispiel
MemoTextnotizen hinterlassen„Heute war das Wetter schön“
ZeitstempelZeiterfassung (Start/Ende)„Besprechung 10:00–11:30“
AufgabeAufgaben verwalten„Bericht schreiben“
AusgabeGeldausgaben erfassen„Supermarkt: Brot 2,50 €“
StimmungStimmung als Wert von 0–10 erfassen„7“
ZahlenwertNumerische Daten erfassen„Gewicht: 65,5“
LesezeichenURLs speichern„https://example.com“
DateiBilder und Dateien hochladenFotos, PDFs usw.

Schritte zum Hinzufügen eines Eintrags

  1. Tippen Sie auf die Schaltfläche „+“ unten rechts auf dem Bildschirm.
  2. Wählen Sie den gewünschten Eintragstyp aus.
  3. Füllen Sie das Eingabeformular aus.
  4. Drücken Sie die Schaltfläche „Speichern“, um den Eintrag zu sichern.
Tipp: Mit dem Notizblock-Bildschirm können Sie mehrere Einträge als Text auf einmal eingeben. Weitere Informationen finden Sie im folgenden Abschnitt „3. Mehrere Einträge mit dem Notizblock auf einmal eingeben“.

3. Mehrere Einträge mit dem Notizblock auf einmal eingeben

Auf dem Notizblock-Bildschirm können Sie durch einfaches Schreiben von Text mehrere Einträge gleichzeitig erstellen.

Grundlegende Verwendung

  1. Öffnen Sie den Notizblock-Bildschirm (wählen Sie „Notizblock“ aus dem Menü der Schaltfläche „+“).
  2. Geben Sie den zu erfassenden Inhalt in das Textfeld ein.
  3. Drücken Sie die Schaltfläche „Speichern“ (alternativ können Sie ! eingeben und absenden).

Eingabe mit Vorlagen

Mit Vorlagen können Sie Standardtexte mit einem Fingertipp eingeben.

  1. Wählen Sie auf dem Notizblock-Bildschirm eine Vorlage aus der Vorlagenliste aus.
  2. Der Inhalt der Vorlage wird in das Textfeld eingefügt.
  3. Bearbeiten Sie den Inhalt bei Bedarf und speichern Sie ihn.
Tipp: Es ist praktisch, häufig verwendete Erfassungsmuster vorab als Vorlagen zu registrieren. Sie können diese über „Vorlagenstruktur“ im Einstellungsbildschirm verwalten.

4. Einträge anzeigen und suchen

Auf dem Lebensprotokoll-Ansichtsbildschirm können Sie vergangene Einträge als Liste anzeigen und durchsuchen.

Grundlegende Bedienung

  1. Öffnen Sie den Lebensprotokoll-Ansichtsbildschirm über die Navigation am oberen Bildschirmrand.
  2. Die Einträge werden in chronologischer Reihenfolge angezeigt.
  3. Durch Ändern des Datums können Sie Einträge eines bestimmten Tages anzeigen.

Suche und Filterung

  • Nach Tags filtern: Sie können nur Einträge mit einem bestimmten Tag anzeigen.
  • Datumsbereich: Sie können den Zeitraum durch Angabe von Start- und Enddatum eingrenzen.
  • Textsuche: Sie können nach Einträgen suchen, die ein Schlüsselwort enthalten.
Tipp: Wenn Sie die „Anzahl der Lebensprotokoll-Ansichtstage“ in den Einstellungen verkürzen, wird das Laden schneller.

5. Aufgaben verwalten

Auf dem Aufgabenbildschirm können Sie Aufgaben im Kanban-Board-Format verwalten.

Aufgaben erstellen

  1. Öffnen Sie den Aufgabenbildschirm.
  2. Fügen Sie eine Aufgabe über die Schaltfläche „+“ hinzu.
  3. Geben Sie Titel, Board-Name, Fälligkeitsdatum usw. ein.
  4. Nach dem Speichern wird die Aufgabe auf dem angegebenen Board angezeigt.

Aufgaben abschließen

  1. Tippen Sie auf eine erledigte Aufgabe, um die Details zu öffnen.
  2. Setzen Sie ein Häkchen, um die Aufgabe als abgeschlossen zu markieren.
Tipp: Erstellen Sie mehrere Boards, um Aufgaben nach Kategorien wie „Arbeit“, „Privat“ oder „Einkaufen“ zu organisieren.

6. Einstellungen ändern

Über den Einstellungsbildschirm können Sie das Verhalten der Anwendung anpassen.

Einstellungsbildschirm öffnen

  1. Tippen Sie auf das Zahnradsymbol in der Symbolleiste am oberen Bildschirmrand.
  2. Der Einstellungsbildschirm wird geöffnet.

Häufig verwendete Einstellungen

EinstellungBeschreibung
SpracheÄndert die Anzeigesprache der Benutzeroberfläche.
Dunkles DesignWechselt zu einem dunklen Farbschema.
StartbildschirmWählt die Seite aus, die nach der Anmeldung zuerst angezeigt wird.
Bildspaltenanzahl der Lebensprotokoll-AnsichtPasst die Anzahl der Spalten für Bild-Thumbnails an.
Tipp: Drücken Sie nach dem Ändern der Einstellungen die Schaltfläche „Übernehmen“. Durch Drücken von „Abbrechen“ werden die Änderungen verworfen.

8. Vorlagen nutzen

Wenn Sie häufig verwendete Notizblock-Syntaxmuster als Vorlagen speichern, sparen Sie sich die wiederholte Eingabe.

Vorlagenverwaltung

  1. Öffnen Sie den Einstellungsbildschirm.
  2. Klicken Sie auf die Schaltfläche „Vorlagenstruktur“.
  3. Sie können Vorlagen hinzufügen, bearbeiten und löschen.

Beispiel für die Vorlagenerstellung

Zum Beispiel, um eine tägliche Tagebuchvorlage zu erstellen:

。Tagebuch
、
ーら

Mit dieser Vorlage wird auf dem Notizblock-Bildschirm mit einem Fingertipp die Tagebuchvorlage eingefügt, und Sie müssen nur noch den Memoinhalt und den Stimmungswert ergänzen.

Tipp: Verwenden Sie Ordner, um Vorlagen nach Kategorien zu organisieren und sie leichter zu finden.

9. Zeit mit Zeitstempeln erfassen

Mit der Zeitstempel-Funktion können Sie Arbeitszeiten und Aktivitätszeiten genau erfassen.

Grundlegende Zeitstempel-Bedienung

  1. Start: Verwenden Sie das Präfix ーた im Notizblock oder fügen Sie einen Zeitstempel über das Menü „+“ hinzu. Geben Sie Titel und Startzeit an.
  2. Ende: Verwenden Sie das Präfix ーえ im Notizblock oder geben Sie die Endzeit im Bearbeitungsbildschirm des Zeitstempels ein. Der Zeitstempel mit dem passenden Titel wird beendet.

Laufende Zeitstempel verwalten

Auf dem Zeitstempel-Listenbildschirm können Sie aktuell laufende Zeitstempel einsehen.

  • Laufende Zeitstempel werden in einer Liste angezeigt.
  • Durch Antippen können Sie die Endzeit festlegen.
Tipp: Mit der Notizblock-Syntax ーいえ tritt kein Fehler auf, auch wenn kein passender Zeitstempel vorhanden ist. Das Beenden ist ohne Existenzprüfung sicher möglich.

10. Daten in der Aggregationsansicht analysieren

Auf dem Aggregationsansichts-Bildschirm können Sie erfasste Daten in Diagrammen und Zusammenfassungstabellen visualisieren.

Verwendung der Aggregationsansicht

  1. Öffnen Sie den Aggregationsansichts-Bildschirm über die Navigation.
  2. Wählen Sie Art und Zeitraum der Aggregation aus.
  3. Die Daten werden in Diagrammen und Tabellen angezeigt.

Aggregationseinstellungen

Über die Schaltfläche „Aggregationsansicht“ im Einstellungsbildschirm können Sie Aggregationsmethoden und die Ansichtskonfiguration verwalten.

Tipp: Wenn Sie die Entwicklung von Stimmungswerten oder Ausgabedaten im Zeitverlauf betrachten, werden Lebensmuster sichtbar.

11. Externe Integrationen nutzen

URLs mit Bookmarklet erfassen

Wenn Sie ein Bookmarklet in der Lesezeichenleiste Ihres Browsers registrieren, können Sie die aktuell angezeigte Webseite mit einem Klick in gkill speichern.

  1. Kopieren Sie die unter „Bookmarklet-Adresse“ im Einstellungsbildschirm angezeigte URL.
  2. Registrieren Sie diese als neues Lesezeichen in der Lesezeichenleiste Ihres Browsers.
  3. Klicken Sie auf das registrierte Lesezeichen, während Sie die zu speichernde Webseite geöffnet haben.

GPS-Aufzeichnung und Kartenanzeige

Wenn GPS-Protokolle aufgezeichnet werden, können Sie den Bewegungsverlauf auf einer Karte anzeigen.

  1. Konfigurieren Sie den „Google Maps API-Schlüssel“ im Einstellungsbildschirm.
  2. Wenn GPS-Daten aufgezeichnet sind, steht die Kartenanzeige in der Lebensprotokoll-Ansicht zur Verfügung.
Tipp: Erhalten Sie den Google Maps API-Schlüssel, indem Sie die Maps JavaScript API in der Google Cloud Console aktivieren.

12. Datensicherung und -verwaltung

Serverkonfiguration (für Administratoren)

Wenn Sie sich mit einem Administratorkonto anmelden, wird im Einstellungsbildschirm die Schaltfläche „Serverkonfiguration“ angezeigt.

  • Portnummer: Sie können den Port ändern, auf dem der Server lauscht (Standard: 9999).
  • TLS-Einstellungen: Sie können HTTPS-Kommunikation aktivieren.
  • Zugriffsbeschränkung: Sie können den Zugriff auf lokale Verbindungen beschränken.

Kontoverwaltung

Administratoren können auf dem Bildschirm „Kontoverwaltung“ Benutzerkonten hinzufügen und löschen.

Datensicherung

Alle Daten von gkill werden im Home-Verzeichnis gespeichert (Standard: ~/gkill). Durch Sichern dieses Verzeichnisses können alle Daten bewahrt werden.

Tipp: Es wird empfohlen, das Home-Verzeichnis regelmäßig zu sichern. Besonders wichtig sind die Verzeichnisse datas/ und configs/.
