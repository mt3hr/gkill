AI 集成

通过将 Claude 等 AI 助手与 gkill 连接，您可以通过自然对话搜索、添加和编辑生活日志记录。

可以做的事情

操作示例
搜索记录"显示上周的午餐笔记" / "汇总本月支出"
添加记录"添加今天的备忘录" / "创建一个任务"
更新 / 删除记录"将此任务标记为已完成"
获取 GPS / 文件"显示今天的路线"
查看设置"列出我的模板"

集成类型

类型说明
只读AI 只能查看和搜索记录，不能添加或修改。
只写AI 只能添加、更新和删除记录。
读写AI 可以查看、添加、更新和删除记录。

设置方法（Claude Desktop）

在 Claude Desktop 配置文件（claude_desktop_config.json）中添加以下内容。

只读集成

{
  "mcpServers": {
    "gkill": {
      "command": "npx",
      "args": ["-y", "gkill-mcp-server"],
      "env": {
        "GKILL_BASE_URL": "http://localhost:9999",
        "GKILL_USER": "您的用户ID",
        "GKILL_PASSWORD_SHA256": "密码的SHA256哈希值"
      }
    }
  }
}
提示： GKILL_PASSWORD_SHA256 是密码的 SHA256 哈希值。可以使用操作系统提供的工具生成。

设置方法（Claude Code）

npx @anthropic-ai/claude-code mcp add gkill \
  --command npx \
  --args "-y,gkill-mcp-server" \
  --env GKILL_BASE_URL=http://localhost:9999 \
  --env GKILL_USER=您的用户ID \
  --env GKILL_PASSWORD_SHA256=哈希值

远程访问

使 gkill 服务器可通过互联网访问，并将 GKILL_BASE_URL 设置为其公共地址。出于安全考虑，强烈建议启用 TLS（HTTPS）。

提示： TLS 设置可从服务器设置中配置。
