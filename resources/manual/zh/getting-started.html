入门指南

gkill是一款用于记录和浏览日常生活日志的应用程序。

gkill的功能

记录类型 说明
备忘录 记录文本备忘录
打卡 记录开始和结束时间
任务 使用看板管理待办事项
心情 用0到10的数值记录心情
支出 记录支出金额和内容
书签 保存网页URL
数值记录 记录任意数值数据
文件 上传并记录图片或文档

启动方式

环境启动方式
Windows运行 gkill.exe 或 gkill_server.exe
浏览器 / PWA启动 gkill_server，在浏览器中访问 http://localhost:9999
Android安装APK并启动应用

首次账户注册

  1. 首次启动gkill时，账户注册页面会自动显示。
  2. 输入用户ID（这将成为您的账户名称）。
  3. 输入密码，然后再次输入进行确认。
  4. 输入管理员密码并再次确认（这是为自动创建的 admin 账户设置密码，用于服务器管理）。
  5. 点击"注册"按钮创建账户。
  6. 页面将重定向到登录页面，使用刚创建的账户登录。
提示: admin 账户在服务器首次启动时自动创建。此页面同时设置 admin 账户的密码和创建您的普通用户账户。

登录后，将显示默认页面（默认为生活日志视图）。请在设置页面中配置语言和暗色主题等基本设置。

60秒内创建您的第一条记录

  1. 登录。
  2. 点击屏幕右下角的"+"按钮。
  3. 选择"记事本"。
  4. 在文本区域中输入您想要记录的内容（例如：今天午餐吃了咖喱）。
  5. 按"保存"按钮即可创建记录。
  6. 从导航菜单打开"生活日志视图"，即可看到您输入的记录。
提示： 这就是创建第一条记录所需的全部操作！学会记事本语法后，您可以用同样的方式记录任务、心情值等。详情请参阅教程。
提示： 点击"+"按钮打开菜单，长按则直接打开记事本。
