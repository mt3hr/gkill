应用设置

管理应用程序全局设置和数据结构的界面。您可以调整显示偏好、行为选项和外部集成。

顶部操作按钮

按钮 说明
服务器设置（仅管理员） 打开服务器配置界面，包括端口、TLS、访问限制等设置。仅对管理员帐户可见。
重新加载 从服务器重新加载数据。长按可执行强制重新加载。
注销 结束当前会话并注销。

常规设置

设置项 类型 说明
语言 下拉菜单 选择界面语言：ja（日语）、en（英语）、zh（中文）、ko（韩语）、es（西班牙语）、fr（法语）、de（德语）。
深色主题 复选框 切换深色模式的开启/关闭。适合在暗光环境下使用。
热重载 复选框 启用后，当检测到数据更改时，生活日志视图会自动刷新。
列表中显示标签 复选框 控制是否在记录列表中显示标签。
显示分享页脚 复选框 控制是否在页脚区域显示分享选项。

显示设置

设置项 类型 说明
生活日志视图显示天数 复选框 + 数值 设置生活日志视图中显示的记录天数。设为-1表示无限制。较短的期间加载更快。
默认页面 下拉菜单 选择登录后显示的初始页面。
生活日志视图图片列数 数值（1-10） 设置生活日志视图中缩略图网格的列数。请根据屏幕大小进行调整。
任务默认看板 下拉菜单 + 添加按钮 设置任务界面默认显示的看板。可使用添加按钮创建新看板。

外部集成

设置项 类型 说明
书签小工具地址 只读 用于浏览器书签栏的JavaScript书签小工具URL。可从任何网页一键保存书签到gkill。
Google Maps API密钥 文本输入 输入Google Maps API密钥以启用GPS地图功能。地图视图需要此密钥才能运行。

数据结构管理

这些按钮可打开专用编辑器，用于管理数据的组织方式。

按钮 说明
标签结构 管理标签层次结构和标签分组。可创建、重命名、重新排序标签，并设置父子关系。
存储库结构 管理数据存储位置和存储库。可添加、删除或配置记录的存储位置。
配置文件结构 管理设备和配置文件设置。可编辑每台设备的配置。
记录类型结构 管理各种记录类型存储在哪些存储库中。
模板结构 管理备忘录文本模板，用于快速输入。可创建和编辑常用文本模板。
汇总视图 管理数据汇总方式和汇总视图配置。
关联信息 管理关联信息结构。

底部操作按钮

按钮 说明
应用 保存所有更改。
取消 放弃更改并关闭界面。

页脚

界面底部显示应用程序信息：

  • 版本号 — 可点击链接到项目页面。
  • 构建时间戳 — 应用程序的构建日期和时间。
  • 提交哈希 — 构建所基于的Git提交。
提示：更改设置后，请按"应用"按钮保存。按"取消"将放弃所有更改。
提示：智能手机上建议将图片列数设为1-2列，电脑上设为3-5列效果更佳。
提示：要使用GPS地图功能，需要在Google Cloud Console中启用Maps JavaScript API并输入API密钥。
