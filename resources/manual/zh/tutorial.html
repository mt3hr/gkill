教程

本教程分为基础篇和进阶篇，介绍gkill的使用方法。初次使用的用户请从基础篇开始依次阅读。

关于此对话框： 此帮助对话框允许鼠标点击和触摸操作穿透到背后的页面。您可以在保持对话框显示的同时操作背后的应用程序界面。
在智能手机上使用： gkill可作为PWA（渐进式Web应用）运行。从浏览器菜单中选择"添加到主屏幕"（iOS Safari）或"安装应用"（Android Chrome），即可像原生应用一样从主屏幕直接启动。

术语说明

在gkill中，数据的种类称为"记录类型"（备忘录、打卡、任务等），操作的页面称为"页面名"（记事本、打卡记事本、任务等）。例如，"备忘录"是指文本备忘这种记录类型，而"记事本"是指输入该记录的页面。"+"按钮菜单中显示的"记事本"、"打卡"等都是用于添加各类记录的页面名。

切换页面

点击页面顶部工具栏中显示的页面标题（如"生活日志视图"、"任务"等），将打开下拉菜单，可以导航到各个页面。

任务看板形式的任务管理

基础篇

1. 登录gkill

首次启动时

首次启动gkill时，将自动显示账户注册页面。

  1. 输入用户ID（这将成为您的账户名称）。
  2. 输入密码，然后再次输入以确认。
  3. 设置管理员密码（用于服务器管理）。
  4. 点击"注册"按钮，账户即创建完成。
  5. 页面将重定向到登录页面，请使用刚创建的账户登录。

常规登录

  1. 在用户ID输入框中输入账户的用户ID。
  2. 在密码输入框中输入密码。
  3. 点击"登录"按钮，或按回车键。
提示： 登录后的会话有效期为30天。在到期之前无需重新登录。

2. 添加记录

gkill可以记录各种类型的生活日志。通过页面右下角的"+"按钮来添加记录。

记录类型用途输入示例
备忘录留下文本备忘"今天天气很好"
打卡时间记录（开始/结束）"会议 10:00~11:30"
任务管理待办事项"写报告"
支出记录消费情况"便利店：饭团 150日元"
心情用0~10的数值记录心情"7"
数值记录数值数据"体重：65.5"
书签保存URL"https://example.com"
文件上传图片或文件照片、PDF等

添加记录的步骤

  1. 点击页面右下角的"+"按钮。
  2. 选择要添加的记录类型。
  3. 在输入表单中填写内容。
  4. 点击"保存"按钮保存记录。
提示： 使用记事本页面，可以通过文本批量输入多条记录。详情请参阅后文"3. 使用记事本批量输入多条记录"。

3. 使用记事本批量输入多条记录

在记事本页面中，只需输入文本即可一次性创建多条记录。

基本使用方法

  1. 打开记事本页面（从"+"按钮的菜单中选择"记事本"）。
  2. 在文本区域输入要记录的内容。
  3. 点击"保存"按钮（也可以输入 ！ 后发送）。

使用模板输入

使用模板可以一键输入固定格式的文本。

  1. 在记事本页面从模板列表中选择一个模板。
  2. 模板内容将被插入到文本区域中。
  3. 根据需要编辑内容后保存。
提示： 将常用的记录模式预先注册为模板会非常方便。可以在设置页面的"模板结构"中进行管理。

4. 查看和搜索记录

在生活日志视图页面中，可以列表显示和搜索过去的记录。

基本操作

  1. 从页面顶部的导航栏打开生活日志视图页面。
  2. 记录将按日期顺序列表显示。
  3. 更改日期可以查看特定日期的记录。

搜索和筛选

  • 按标签筛选： 可以仅显示带有特定标签的记录。
  • 日期范围： 指定开始日期和结束日期来筛选时间段。
  • 文本搜索： 搜索包含关键词的记录。
提示： 在设置页面中缩短"生活日志视图显示天数"可以加快加载速度。

5. 管理任务

在任务页面中，可以使用看板形式管理任务。

创建任务

  1. 打开任务页面。
  2. 通过"+"按钮添加任务。
  3. 输入标题、看板名称、截止日期等信息。
  4. 保存后，任务将显示在指定的看板中。

完成任务

  1. 点击已完成的任务打开详情。
  2. 勾选复选框将任务标记为已完成。
提示： 创建多个看板，按"工作""个人""购物"等类别分类管理，可以更方便地整理任务。

6. 更改设置

可以通过设置页面自定义应用程序的行为。

打开设置页面

  1. 点击页面顶部工具栏中的齿轮图标。
  2. 设置页面将会打开。

常用设置

设置项说明
语言更改UI的显示语言。
深色主题切换为暗色配色方案。
启动页面选择登录后首先显示的页面。
生活日志视图图片列数调整图片缩略图的列数。
提示： 更改设置后请点击"应用"按钮。点击"取消"将放弃更改。

8. 活用模板

将常用的记事本语法模式保存为模板，可以省去每次输入的麻烦。

模板管理

  1. 打开设置页面。
  2. 点击"模板结构"按钮。
  3. 可以进行模板的添加、编辑和删除。

模板创建示例

例如，创建每日日记模板的情况：

使用此模板，在记事本页面即可一键输入日记框架，只需追加备忘录内容和心情值即可完成记录。

提示： 使用文件夹按类别整理模板，可以更容易找到所需模板。

9. 使用打卡记录时间

页面名 用途
打卡记事本 带时间戳输入备忘录的页面。也可以上传文件。
进行中 当前活动的打卡会话列表。可以在此结束会话。
记事本 通过记事本语法以文本方式开始和结束打卡。可与其他记录一起批量输入。

使用打卡功能，可以精确记录工作时间和活动时间。

打卡的基本操作

  1. 开始： 在记事本中使用 ーた 前缀，或从"+"菜单添加打卡记录。指定标题和开始时间。
  2. 结束： 在记事本中使用 ーえ 前缀，或在打卡编辑页面输入结束时间。标题匹配的打卡将被结束。

管理进行中的打卡

在打卡列表页面中，可以确认当前正在进行的打卡。

  • 正在进行的打卡将以列表显示。
  • 点击可以设置结束时间。
提示： 使用记事本语法的 ーいえ，即使没有匹配的打卡也不会报错。可以安全地结束打卡而无需确认是否存在。

10. 在汇总视图中分析数据

生活日志视图画面内的汇总视图可以通过汇总表可视化记录的数据。

汇总视图的使用方法

  1. 点击生活日志视图工具栏中的汇总图标，显示汇总视图。
  2. 选择汇总类型和时间段。
  3. 数据将以图表或表格形式显示。

汇总设置

可以从设置页面的"汇总视图"按钮管理汇总方法和视图配置。

提示： 按时间序列查看心情值和支出数据的变化趋势，可以发现生活规律。

11. 利用外部集成

使用书签工具记录URL

将书签工具注册到浏览器的书签栏中，可以一键将正在浏览的网页保存到gkill。

  1. 复制设置页面中"书签工具地址"显示的URL。
  2. 将其作为新书签注册到浏览器的书签栏中。
  3. 在打开想要保存的网页时，点击已注册的书签即可。

GPS记录和地图显示

如果正在记录GPS日志，可以在地图上显示移动轨迹。

  1. 在设置页面中设置"Google Maps API密钥"。
  2. 如果已记录GPS数据，则可以在生活日志视图中使用地图显示功能。
提示： Google Maps API密钥需要在Google Cloud Console中启用Maps JavaScript API后获取。

12. 共享记录

在gkill中，您可以生成共享链接与他人共享记录。

生成共享链接的步骤

  1. 打开要共享的记录的详细视图或右键菜单。
  2. 选择"共享"。
  3. 将生成共享URL。根据用途使用相应的URL：
URL类型用途
本地URL同一设备内访问（localhost）
局域网URL同一网络内其他设备访问
广域网URL通过互联网进行外部访问

共享数据控制

共享时，您可以选择包含哪些数据。

选项说明
标签在共享中包含记录的标签
文本在共享中包含记录的文本内容
打卡在共享中包含打卡时间信息
位置信息在共享中包含GPS位置信息
仅时间仅在共享中包含记录的时间信息
提示: 要使用局域网URL或广域网URL，需要在应用程序设置中配置服务器的IP地址。

致共享链接的接收者

打开共享链接后，即使没有gkill账户也可以查看记录。

  • 显示的记录范围由链接创建者的设置决定。
  • 无法编辑或删除记录（仅限查看）。
  • 如果链接已失效，请联系发送者。

13. 数据备份与管理

服务器设置（管理员）

使用管理员账户登录后，设置页面中将显示"服务器设置"按钮。

  • 端口号： 可以更改服务器监听的端口（默认值：9999）。
  • TLS设置： 可以启用HTTPS通信。
  • 访问限制： 可以限制为仅本地访问。

账户管理

管理员可以在"账户管理"页面中添加、启用和禁用用户账户。

数据备份

gkill的所有数据都保存在主目录中（默认值：~/gkill）。备份此目录即可保全所有数据。

提示： 建议定期备份主目录。其中 datas/ 和 configs/ 目录尤为重要。
