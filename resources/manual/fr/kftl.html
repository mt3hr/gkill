Bloc-notes

Ecran de bloc-notes pour enregistrer des evenements de la vie via la syntaxe du bloc-notes. Le bloc-notes utilise un format de texte structure par lignes permettant la saisie par lots de divers enregistrements.

Boutons de la barre d'outils

Bouton Description
⚙ Parametres Ouvre l'ecran des parametres de l'application.
❔ AideAffiche l'aide de l'écran actuel.

Elements de l'ecran

Element Description
Zone de texte Zone de saisie du contenu des enregistrements en syntaxe du bloc-notes.
Bouton de modele Insere un texte de modele de bloc-notes predefini.
Bouton d'enregistrement Cliquez sur le bouton d'enregistrement pour analyser et enregistrer le texte du bloc-notes saisi.

A propos du bloc-notes

Le bloc-notes utilise un format de texte propre a gkill qui permet de creer en masse plusieurs enregistrements dans un seul texte. Chaque ligne represente une commande ou une entree de donnees.

Astuce : En utilisant le bouton de modele, vous pouvez inserer rapidement des formats de saisie couramment utilises sans avoir a memoriser la syntaxe du bloc-notes.

FAB (Bouton d'action flottant) "+"

En cliquant sur le bouton "+", un menu se deploie pour creer rapidement des enregistrements :

TypeDescription
MémoCree une note textuelle.
Bloc-notes d'entrées de tempsCree une note avec horodatage.
Enregistrements numériquesEnregistre des donnees numeriques.
SignetsEnregistre un signet URL.
Suivi du tempsDemarre une session de suivi du temps.
TâchesCree une nouvelle tache.
DépensesEnregistre des informations de depense.
HumeurEnregistre une valeur d'humeur (0-10).
TéléverserTeleverse un fichier.
Astuce : Un appui long sur le bouton "+" ouvre directement l'ecran de saisie du bloc-notes, permettant une saisie par lots rapide.
Raccourci : Appuyez sur Entrée sur cet écran pour ouvrir le bloc-notes. C'est pratique pour ajouter rapidement des enregistrements.
