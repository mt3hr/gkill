Tutoriel

Ce guide explique comment utiliser gkill, en deux parties : les bases et les fonctions avancees. Si vous debutez, veuillez lire la partie basique en premier.

A propos de cette boite de dialogue : Cette boite de dialogue d'aide laisse passer les clics de souris et les interactions tactiles vers l'ecran situe en arriere-plan. Vous pouvez interagir avec l'application tout en gardant cette boite de dialogue ouverte.
Utilisation sur smartphone : gkill fonctionne comme une PWA (Progressive Web App). Selectionnez « Ajouter a l'ecran d'accueil » (iOS Safari) ou « Installer l'application » (Android Chrome) depuis le menu de votre navigateur pour le lancer directement depuis votre ecran d'accueil.

Terminologie

Dans gkill, les types de donnees sont appeles « types d'enregistrement » (Memo, Horodatage, Tache, etc.) et les ecrans de travail sont appeles « noms d'ecran » (Bloc-notes, Bloc-notes d'horodatage, Taches, etc.). Par exemple, « Memo » designe le type de donnee textuelle, tandis que « Bloc-notes » designe l'ecran de saisie. Les elements affiches dans le menu du bouton « + » sont des noms d'ecran pour ajouter chaque type d'enregistrement.

Partie basique

1. Se connecter a gkill

Premier lancement

Lors du premier lancement de gkill, l'ecran d'inscription s'affiche automatiquement.

  1. Saisissez un identifiant utilisateur (ce sera le nom de votre compte).
  2. Saisissez un mot de passe, puis confirmez-le en le saisissant une seconde fois.
  3. Saisissez un mot de passe administrateur et confirmez-le (il s'agit du mot de passe du compte admin cree automatiquement, utilise pour la gestion du serveur).
  4. Appuyez sur le bouton "Inscription" pour creer votre compte.
  5. Vous serez redirige vers l'ecran de connexion. Connectez-vous avec le compte que vous venez de creer.

Connexion habituelle

  1. Saisissez l'identifiant de votre compte dans le champ prevu a cet effet.
  2. Saisissez votre mot de passe dans le champ correspondant.
  3. Cliquez sur le bouton "Connexion" ou appuyez sur la touche Entree.
Astuce : La session reste active pendant 30 jours apres la connexion. Vous n'avez pas besoin de vous reconnecter avant l'expiration de ce delai.

2. Ajouter un enregistrement

gkill permet d'enregistrer differents types de journaux de vie. Ajoutez un enregistrement en appuyant sur le bouton "+" en bas a droite de l'ecran.

Type d'enregistrementUsageExemple de saisie
MemoLaisser une note textuelle"Il faisait beau aujourd'hui"
PointageEnregistrer le temps (debut/fin)"Reunion 10h00-11h30"
TacheGerer les choses a faire"Rediger le rapport"
DepenseEnregistrer les depenses"Superette : sandwich 3,50 EUR"
HumeurEnregistrer l'humeur sur une echelle de 0 a 10"7"
Enregistrements numériquesEnregistrer des donnees numeriques"Poids : 65,5"
SignetSauvegarder une URL"https://example.com"
FichierTelecharger des images ou fichiersPhotos, PDF, etc.

Procedure d'ajout d'un enregistrement

  1. Appuyez sur le bouton "+" en bas a droite de l'ecran.
  2. Selectionnez le type d'enregistrement que vous souhaitez ajouter.
  3. Remplissez le formulaire de saisie.
  4. Appuyez sur le bouton "Enregistrer" pour sauvegarder.
Astuce : En utilisant l'ecran du bloc-notes, vous pouvez saisir plusieurs enregistrements a la fois sous forme de texte. Pour plus de details, consultez la section "3. Saisir plusieurs enregistrements a la fois avec le bloc-notes" ci-dessous.

3. Saisir plusieurs enregistrements a la fois avec le bloc-notes

L'ecran du bloc-notes permet de creer plusieurs enregistrements simultanement en ecrivant simplement du texte.

Utilisation de base

  1. Ouvrez l'ecran du bloc-notes (selectionnez "Bloc-notes" dans le menu du bouton "+").
  2. Saisissez le contenu que vous souhaitez enregistrer dans la zone de texte.
  3. Appuyez sur le bouton "Enregistrer" (vous pouvez egalement saisir ! et envoyer).

Saisie a l'aide de modeles

Les modeles vous permettent de saisir des textes predefinis en un seul geste.

  1. Sur l'ecran du bloc-notes, selectionnez un modele dans la liste des modeles.
  2. Le contenu du modele est insere dans la zone de texte.
  3. Modifiez le contenu si necessaire, puis enregistrez.
Astuce : Il est pratique d'enregistrer a l'avance les schemas d'enregistrement que vous utilisez frequemment en tant que modeles. Vous pouvez les gerer depuis "Structure des modeles" dans l'ecran de configuration.

4. Consulter et rechercher des enregistrements

L'ecran de visualisation des journaux de vie vous permet d'afficher et de rechercher vos enregistrements passes.

Operations de base

  1. Ouvrez l'ecran de visualisation des journaux de vie depuis la barre de navigation en haut de l'ecran.
  2. Les enregistrements sont affiches par ordre chronologique.
  3. Changez la date pour consulter les enregistrements d'un jour specifique.

Recherche et filtrage

  • Filtrer par tag : Vous pouvez afficher uniquement les enregistrements portant un tag specifique.
  • Plage de dates : Vous pouvez restreindre la periode en specifiant une date de debut et de fin.
  • Recherche textuelle : Vous pouvez rechercher les enregistrements contenant un mot-cle.
Astuce : Reduire le "nombre de jours affiches dans la vue des journaux de vie" dans l'ecran de configuration accelere le chargement.

5. Gerer les taches

L'ecran des taches permet de gerer vos taches sous forme de tableau Kanban.

Creation d'une tache

  1. Ouvrez l'ecran des taches.
  2. Ajoutez une tache avec le bouton "+".
  3. Saisissez le titre, le nom du tableau, la date limite, etc.
  4. Une fois enregistree, la tache apparait dans le tableau specifie.

Achevement d'une tache

  1. Appuyez sur la tache terminee pour ouvrir les details.
  2. Cochez la case pour marquer la tache comme terminee.
Astuce : Creez plusieurs tableaux pour organiser par categorie, comme "Travail", "Personnel", "Courses", etc.

6. Modifier les parametres

Vous pouvez personnaliser le comportement de l'application depuis l'ecran de configuration.

Comment ouvrir l'ecran de configuration

  1. Appuyez sur l'icone d'engrenage dans la barre d'outils en haut de l'ecran.
  2. L'ecran de configuration s'ouvre.

Parametres frequemment utilises

ParametreDescription
LangueChange la langue d'affichage de l'interface.
Theme sombreBascule vers un theme aux couleurs sombres.
Ecran au demarrageSelectionne la page affichee en premier apres la connexion.
Nombre de colonnes d'images dans la vue des journaux de vieAjuste le nombre de colonnes des miniatures d'images.
Astuce : Apres avoir modifie les parametres, appuyez sur le bouton "Appliquer". Si vous appuyez sur "Annuler", les modifications seront ignorees.

8. Exploiter les modeles

En sauvegardant les schemas de syntaxe du bloc-notes que vous utilisez frequemment en tant que modeles, vous evitez de les saisir a chaque fois.

Gestion des modeles

  1. Ouvrez l'ecran de configuration.
  2. Cliquez sur le bouton "Structure des modeles".
  3. Vous pouvez ajouter, modifier ou supprimer des modeles.

Exemple de creation de modele

Par exemple, pour creer un modele de journal quotidien :

.journal
,
-ra

En utilisant ce modele, l'ebauche du journal est saisie en un seul geste sur l'ecran du bloc-notes, et il vous suffit d'ajouter le contenu du memo et la valeur d'humeur pour completer l'enregistrement.

Astuce : Utilisez des dossiers pour organiser vos modeles par categorie afin de les retrouver plus facilement.

9. Enregistrer le temps avec les pointages

Nom de l'écran Usage
Bloc-notes d'horodatage Saisir des notes avec horodatage. Le téléversement de fichiers est également possible.
En cours Liste des sessions de suivi du temps actives. Vous pouvez terminer des sessions ici.
Bloc-notes Démarrer et terminer le suivi du temps par texte avec la syntaxe du bloc-notes. Peut être combiné avec d'autres enregistrements en lot.

La fonction de pointage vous permet d'enregistrer precisement vos temps de travail et d'activite. Le "Bloc-notes d'horodatage" est un ecran dedie combinant saisie de notes et gestion des sessions en cours, tandis que le "Bloc-notes" permet de demarrer/terminer le suivi du temps via des prefixes avec d'autres enregistrements en lot.

Operations de base du pointage

  1. Debut : Utilisez le prefixe -ta dans le bloc-notes ou ajoutez un pointage depuis le menu "+". Specifiez le titre et l'heure de debut.
  2. Fin : Utilisez le prefixe -e dans le bloc-notes ou saisissez l'heure de fin dans l'ecran de modification du pointage. Le pointage dont le titre correspond sera cloture.

Gerer les pointages en cours

L'ecran de liste des pointages vous permet de consulter les pointages actuellement en cours.

  • Les pointages en cours sont affiches dans une liste.
  • Appuyez dessus pour definir l'heure de fin.
Astuce : En utilisant -ie dans la syntaxe du bloc-notes, aucune erreur ne se produit meme s'il n'y a pas de pointage correspondant. Cela permet de cloturer en toute securite sans verification prealable.

10. Analyser les donnees avec la vue d'agregation

La vue d'agrégation intégrée à la vue journal de vie vous permet de visualiser les données enregistrées sous forme de tableaux récapitulatifs.

Utilisation de la vue d'agregation

  1. Cliquez sur l'icône d'agrégation dans la barre d'outils de la vue journal de vie pour afficher la vue d'agrégation.
  2. Selectionnez le type d'agregation et la periode.
  3. Les donnees sont affichees sous forme de tableaux recapitulatifs.

Configuration de l'agregation

Depuis le bouton "Vue d'agregation" dans l'ecran de configuration, vous pouvez gerer les methodes d'agregation et la composition des vues.

Astuce : Suivre l'evolution de l'humeur et des depenses dans le temps permet de mieux comprendre vos habitudes de vie.

11. Tirer parti des integrations externes

Enregistrer des URL avec un bookmarklet

En ajoutant un bookmarklet a la barre de favoris de votre navigateur, vous pouvez sauvegarder la page Web consultee dans gkill en un seul clic.

  1. Copiez l'URL affichee dans "Bookmarklet" dans l'ecran de configuration.
  2. Ajoutez-la en tant que nouveau favori dans la barre de favoris de votre navigateur.
  3. Lorsque vous consultez une page Web que vous souhaitez sauvegarder, cliquez sur le favori enregistre.

Enregistrement GPS et affichage sur carte

Si vous enregistrez des journaux GPS, vous pouvez afficher votre historique de deplacements sur une carte.

  1. Configurez la "Cle API Google Map" dans l'ecran de configuration.
  2. Si des donnees GPS sont enregistrees, l'affichage cartographique est disponible dans la vue des journaux de vie.
Astuce : Pour obtenir une cle API Google Maps, activez l'API Maps JavaScript dans la console Google Cloud.

12. Partager des enregistrements

Dans gkill, vous pouvez generer des liens de partage pour partager des enregistrements avec d'autres personnes.

Comment generer un lien de partage

  1. Ouvrez la vue detaillee ou le menu contextuel de l'enregistrement que vous souhaitez partager.
  2. Selectionnez "Partager".
  3. Une URL de partage sera generee. Utilisez l'URL appropriee selon votre besoin :
Type d'URLUsage
URL localeAcces depuis le meme appareil (localhost)
URL LANAcces depuis d'autres appareils sur le meme reseau
URL WANAcces externe via Internet

Controle des donnees partagees

Lors du partage, vous pouvez choisir quelles donnees inclure.

OptionDescription
TagsInclure les tags attaches a l'enregistrement
TexteInclure le contenu texte de l'enregistrement
HorodatagesInclure les informations d'horodatage
LocalisationInclure les informations de localisation GPS
Heure uniquementInclure uniquement les informations d'heure
Conseil : Pour utiliser les URLs LAN ou WAN, vous devez configurer l'adresse IP du serveur dans les parametres du serveur.

Pour les destinataires de liens partages

En ouvrant un lien partage, vous pouvez consulter les enregistrements sans avoir de compte gkill.

  • L'etendue des enregistrements affiches est determinee par les parametres du createur du lien.
  • Vous ne pouvez ni modifier ni supprimer les enregistrements (consultation uniquement).
  • Si le lien n'est plus valide, veuillez contacter l'expediteur.

13. Sauvegarde et gestion des donnees

Configuration du serveur (administrateurs)

En vous connectant avec un compte administrateur, le bouton "Configuration du serveur" apparait dans l'ecran de configuration.

  • Numero de port : Vous pouvez modifier le port d'ecoute du serveur (par defaut : 9999).
  • Configuration TLS : Vous pouvez activer la communication HTTPS.
  • Restriction d'acces : Vous pouvez limiter l'acces au reseau local uniquement.

Gestion des comptes

L'administrateur peut ajouter, activer et desactiver des comptes utilisateurs depuis l'ecran "Gestion des comptes".

Sauvegarde des donnees

Toutes les donnees de gkill sont stockees dans le repertoire personnel (par defaut : ~/gkill). Il suffit de sauvegarder ce repertoire pour preserver l'ensemble de vos donnees.

Astuce : Il est recommande de sauvegarder regulierement le repertoire personnel. Les repertoires datas/ et configs/ sont particulierement importants.
