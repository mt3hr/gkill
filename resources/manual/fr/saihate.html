Accueil

Écran permettant d'enregistrer en toute sécurité même depuis l'écran de verrouillage. Il sert de hub pour ajouter rapidement tout type d'enregistrement.

Saihate est un hub léger spécialisé dans l'ajout de nouveaux enregistrements. Il ne propose pas de fonctions de navigation ni d'édition, ce qui élimine tout risque de modifier accidentellement des enregistrements existants. Il fonctionne au sein d'une session authentifiée et n'est accessible qu'après connexion.

Quel écran utiliser : Utilisez Saihate pour noter rapidement un mémo ou un horodatage, la Vue journal de vie pour parcourir et rechercher vos enregistrements, et le Bloc-notes pour saisir des enregistrements en lot via le texte.

Boutons de la barre d'outils

Bouton Description

FAB (Bouton d'action flottant) "+"

En cliquant sur le bouton "+", un menu se deploie pour creer rapidement des enregistrements des 9 types suivants :

Type Description
MemoCree une note textuelle.
Temps+MemoCree une note avec horodatage.
NumeriqueEnregistre des donnees numeriques.
SignetEnregistre un signet URL.
Journal de tempsDemarre une session de suivi du temps.
TacheCree une nouvelle tache.
DepenseEnregistre des informations de depense.
HumeurEnregistre une valeur d'humeur (0-10).
TelechargementTelecharge un fichier.
Astuce : Un appui long sur le bouton "+" ouvre directement l'ecran de saisie du bloc-notes, permettant une saisie par lots rapide.
Astuce : L'écran affiché après la connexion peut être modifié dans le paramètre « Vue par défaut au démarrage » de l'écran de configuration.
Raccourci : Appuyez sur Entrée sur cet écran pour ouvrir le bloc-notes. C'est pratique pour ajouter rapidement des enregistrements.
