Intégration IA

En connectant des assistants IA comme Claude à gkill, vous pouvez rechercher, ajouter et modifier vos journaux de vie par conversation naturelle.

Ce que vous pouvez faire

ActionExemple
Rechercher des enregistrements« Montre-moi mes notes de déjeuner de la semaine dernière » / « Résume les dépenses de ce mois »
Ajouter des enregistrements« Ajoute une note pour aujourd'hui » / « Crée une tâche »
Mettre à jour / supprimer« Marque cette tâche comme terminée »
Récupérer GPS / fichiers« Montre mon itinéraire d'aujourd'hui »
Vérifier la configuration« Liste mes modèles »

Types d'intégration

TypeDescription
Lecture seuleL'IA peut uniquement afficher et rechercher des enregistrements. L'ajout ou la modification n'est pas possible.
Écriture seuleL'IA peut uniquement ajouter, mettre à jour et supprimer des enregistrements.
Lecture et écritureL'IA peut afficher, ajouter, mettre à jour et supprimer des enregistrements.

Configuration (Claude Desktop)

Ajoutez ce qui suit au fichier de configuration de Claude Desktop (claude_desktop_config.json).

Intégration en lecture seule

{
  "mcpServers": {
    "gkill": {
      "command": "npx",
      "args": ["-y", "gkill-mcp-server"],
      "env": {
        "GKILL_BASE_URL": "http://localhost:9999",
        "GKILL_USER": "votre-id-utilisateur",
        "GKILL_PASSWORD_SHA256": "hash-sha256-de-votre-mot-de-passe"
      }
    }
  }
}
Conseil : GKILL_PASSWORD_SHA256 est le hachage SHA256 de votre mot de passe. Vous pouvez le générer avec les outils disponibles sur votre système d'exploitation.

Configuration (Claude Code)

npx @anthropic-ai/claude-code mcp add gkill \
  --command npx \
  --args "-y,gkill-mcp-server" \
  --env GKILL_BASE_URL=http://localhost:9999 \
  --env GKILL_USER=votre-id-utilisateur \
  --env GKILL_PASSWORD_SHA256=votre-hash

Accès à distance

Rendez votre serveur gkill accessible via Internet et définissez GKILL_BASE_URL sur son adresse publique. L'activation de TLS (HTTPS) est fortement recommandée pour la sécurité.

Conseil : Les paramètres TLS peuvent être configurés depuis Paramètres du serveur.
