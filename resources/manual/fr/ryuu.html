Informations associées

Écran de définition des conditions pour rechercher et afficher automatiquement les enregistrements liés à une entrée du journal de vie. Vous pouvez configurer des requêtes par définition pour visualiser les relations entre les enregistrements.

Astuce : Les informations associées sont une sous-vue qui s'affiche lorsque vous sélectionnez un enregistrement dans la vue journal de vie. Elles sont accessibles depuis la vue journal de vie.

Gestion des définitions

Élément Description
Onglets Chaque définition est affichée sous forme d'onglet. Cliquez sur un onglet pour basculer entre les définitions.
Nom Champ de texte permettant de modifier le nom de la définition.
Bouton ＋ (à côté des onglets) Ajoute une nouvelle définition.
Bouton supprimer Supprime la définition actuellement affichée.

Liste des requêtes

Chaque définition affiche une liste de requêtes. Le bouton flottant ＋ permet d'ajouter une nouvelle requête.

Paramètres des requêtes

Paramètre Type Description
Titre Saisie de texte Définit le nom d'affichage de la requête.
Préfixe Saisie de texte Texte affiché avant la valeur correspondante.
Suffixe Saisie de texte Texte affiché après la valeur correspondante.
Groupe de conditions Éditeur de conditions Filtre les cibles de correspondance à l'aide de conditions AND/OR.
Type de correspondance temporelle Liste déroulante Sélectionne la méthode de correspondance temporelle.
Période de recherche (heures) Saisie numérique Spécifie la plage horaire de recherche des enregistrements associés, en heures.
Requête de recherche personnalisée Case à cocher ＋ bouton Activez cette option pour utiliser une requête de recherche personnalisée. Le bouton ouvre l'éditeur de requête.

Types de correspondance temporelle

Type Description
Proche dans le temps Correspond aux enregistrements dont l'heure est la plus proche, avant et après l'enregistrement cible.
Proche dans le temps (avant) Correspond à l'enregistrement dont l'heure est la plus proche avant l'enregistrement cible.
Proche dans le temps (après) Correspond à l'enregistrement dont l'heure est la plus proche après l'enregistrement cible.

Boutons d'édition de requête

Bouton Description
Réinitialiser Annule les modifications en cours.
Enregistrer Enregistre les modifications de la requête.

Boutons d'action inférieurs

Bouton Description
Appliquer Enregistre toutes les modifications des définitions et requêtes.
Annuler Annule les modifications et ferme le dialogue.
Astuce : Les informations associées sont utilisées pour afficher automatiquement les enregistrements liés lors de la consultation du journal de vie.
Astuce : Une période de recherche plus courte ne fait correspondre que les enregistrements temporellement plus proches.
Astuce : Créez plusieurs définitions pour définir différents types de relations.
Astuce : Utilisez les requêtes de recherche personnalisées pour définir des conditions avancées.
