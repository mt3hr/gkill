Parametres de l'application

Ecran de gestion des parametres globaux de l'application et de la structure des donnees. Vous pouvez ajuster les preferences d'affichage, les options de comportement et les integrations externes.

Boutons d'action superieurs

Bouton Description
Parametres du serveur (administrateurs uniquement) Ouvre l'ecran de configuration du serveur pour le port, TLS, les restrictions d'acces, etc. Visible uniquement pour les comptes administrateurs.
Recharger Actualise les donnees depuis le serveur. Appui long pour forcer un rechargement complet.
Deconnexion Termine la session en cours et se deconnecte.

Parametres generaux

Parametre Type Description
Langue Liste deroulante Choisir la langue de l'interface : ja (japonais), en (anglais), zh (chinois), ko (coreen), es (espagnol), fr (francais), de (allemand).
Theme sombre Case a cocher Active ou desactive le mode sombre. Adapte aux environnements peu eclaires.
Rechargement a chaud Case a cocher Lorsqu'il est active, la vue du journal de vie se rafraichit automatiquement lorsque des changements de donnees sont detectes.
Afficher les tags dans la liste Case a cocher Controle si les tags sont affiches sur les fiches d'enregistrement dans les vues en liste.
Afficher le pied de page de partage Case a cocher Controle si les options de partage sont affichees dans la zone du pied de page.

Parametres d'affichage

Parametre Type Description
Periode d'affichage du journal de vie Case a cocher + Nombre Definit le nombre de jours d'enregistrements a afficher dans la vue du journal de vie. -1 pour illimite. Une periode plus courte charge plus rapidement.
Page par defaut Liste deroulante Choisir la page affichee apres la connexion.
Colonnes d'images du journal de vie Nombre (1-10) Definit le nombre de colonnes de la grille de miniatures dans la vue du journal de vie. Ajustez en fonction de la taille de votre ecran.
Tableau de taches par defaut Liste deroulante + Bouton ajouter Definit le tableau par defaut dans l'ecran des taches. Utilisez le bouton d'ajout pour creer de nouveaux tableaux.

Integration externe

Parametre Type Description
Bookmarklet Lecture seule URL du bookmarklet JavaScript pour la barre de favoris du navigateur. Permet d'enregistrer des favoris dans gkill en un clic depuis n'importe quelle page web.
Cle API Google Map Saisie de texte Entrez votre cle API Google Maps pour activer les fonctions de carte GPS. La vue carte necessite cette cle pour fonctionner.

Gestion de la structure des donnees

Ces boutons ouvrent des editeurs dedies pour gerer l'organisation de vos donnees.

Bouton Description
Structure des etiquettes Gere la hierarchie des etiquettes et leurs regroupements. Vous pouvez creer, renommer, reorganiser et organiser les etiquettes en relations parent-enfant.
Structure des dépôts Gere les emplacements de stockage des donnees et les dépôts. Vous pouvez ajouter, supprimer ou configurer les emplacements de stockage des enregistrements.
Structure des profils Gere les configurations des appareils et des profils. Permet de modifier les parametres par appareil.
Structure des types d'enregistrement Gere quels types d'enregistrement sont stockes dans quels depots.
Structure des modeles Gere les modeles de texte du bloc-notes pour la saisie rapide. Vous pouvez creer et modifier des textes predefinies.
Vue de synthese Gere l'agregation des donnees et les configurations des vues de synthese.
Informations associees Gere les structures d'informations associees.

Boutons d'action inferieurs

Bouton Description
Appliquer Enregistre toutes les modifications.
Annuler Annule les modifications et ferme l'ecran.

Pied de page

Le bas de l'ecran affiche les informations de l'application :

  • Numero de version — Lien cliquable vers la page du projet.
  • Horodatage de compilation — Date et heure de compilation de l'application.
  • Hash du commit — Le commit Git a partir duquel la compilation a ete creee.
Astuce : Apres avoir modifie les parametres, appuyez sur "Appliquer" pour enregistrer. Appuyez sur "Annuler" pour annuler toutes les modifications.
Astuce : Pour les smartphones, definissez les colonnes d'images sur 1-2. Pour les ecrans de bureau, 3-5 colonnes convient bien.
Astuce : Pour utiliser les fonctions de carte GPS, activez l'API Maps JavaScript dans Google Cloud Console et entrez la cle API dans Integration externe.
