Enregistrements

Ecran affichant les details d'un enregistrement individuel. Vous pouvez consulter les informations detaillees de differents types d'enregistrements.

Boutons de la barre d'outils

Bouton Description
⚙ Parametres Ouvre l'ecran des parametres de l'application.
❔ AideAffiche l'aide de l'écran actuel.

Types d'enregistrements affiches

Type d'enregistrement Description
Mémo Affiche le contenu des notes textuelles.
Signets Affiche les URLs et titres enregistres.
Taches Affiche l'etat, la date limite et les informations du tableau de la tache.
Humeur Affiche la valeur d'humeur de 0 a 10.
Depenses Affiche le montant et le contenu de la depense.
Journal de temps Affiche l'heure de debut/fin et le temps ecoule.
Fichier Affiche un apercu du fichier telecharge.
Enregistrements numériques Affiche le titre et les données numériques.
Repost Affiche les enregistrements republiés à une autre date/heure.
Journal GPS Affiche les données de journal de localisation GPS.
Journal de commit Git Affiche les informations de commits du dépôt Git.
Notification Affiche les notifications et rappels.

Informations communes affichees

Les informations suivantes sont affichees pour tous les types d'enregistrements :

  • Etiquettes : Liste des etiquettes attribuees a l'enregistrement
  • Temps associe : Informations de date et heure liees a l'enregistrement
  • Dépôt : Nom du dépôt ou les donnees sont stockees

Menu contextuel

Un clic droit (ou un appui long) sur un enregistrement affiche un menu contextuel avec les options suivantes :

  • Modifier
  • Supprimer
  • Partager
  • Ajouter/Modifier une étiquette
  • Ajouter/Modifier un texte
  • Ajouter/Modifier une notification
  • Historique
  • Copier l'ID
Astuce : Vous pouvez ajouter ou modifier des etiquettes directement depuis l'ecran de details de l'enregistrement.
