Configuration du serveur

Cet écran configure le comportement du serveur. Seuls les administrateurs peuvent y accéder.

Bouton supérieur

Appuyez sur le bouton Gestion des comptes pour ouvrir l'écran de gestion des comptes.

Paramètres

ProfilListe déroulante + Ajouter/SupprimerSélectionner le profil actif, en créer un nouveau ou en supprimer un existant

Commande d'ouverture de fichierTexteCommande du système d'exploitation pour ouvrir un fichier

Agent utilisateur des favorisTexteChaîne User-Agent pour la récupération des favoris
Répertoire de données utilisateurTexteChemin vers le répertoire de stockage des données utilisateur

Boutons inférieurs

Appliquer : Enregistrer les paramètres.
Annuler : Annuler les modifications et fermer l'écran.

Les modifications nécessitent un redémarrage du serveur pour prendre effet.
