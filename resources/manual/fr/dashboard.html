Tableau de bord

Une vue complète pour voir tous vos journaux de vie pour la date sélectionnée en un coup d'œil. La moitié supérieure affiche la Vue d'agrégation et la carte GPS de ce jour, et la partie inférieure affiche vos tâches.

Boutons de barre

Bouton Description
← (Jour précédent) Affiche les données du jour précédent.
Étiquette de date (ex: 2025/6/15 Dim) Cliquez pour ouvrir le sélecteur de calendrier.
→ (Jour suivant) Affiche les données du jour suivant.
❔ Aide Affiche l'aide pour l'écran actuel.
⚙ Paramètres Ouvre les paramètres de l'application.
Menu de navigation (clic sur le titre) Permet de naviguer vers chaque écran.

Zones d'affichage

Zone Description
Vue d'agrégation Affiche la Vue d'agrégation du jour sélectionné dans la moitié gauche. Vous pouvez affiner les conditions de recherche dans Paramètres.
Carte GPS Affiche le journal GPS du jour sélectionné sous forme de carte dans la moitié droite.
Liste des tâches Affiche les tâches liées au jour sélectionné en bas. Vous pouvez affiner les conditions de recherche dans Paramètres.

Bouton d'action flottant (+)

Appuyez sur le bouton "+" en bas à droite pour ouvrir le menu d'ajout rapide d'enregistrements.

Élément de menu Contenu Description
Bloc-notes Saisie de texte Ouvre l'écran pour saisir des mémos et divers enregistrements avec la syntaxe du bloc-notes.
Bloc-notes d'horodatage Texte + horodatage Ouvre l'écran pour gérer la saisie de mémos et les sessions d'horodatage simultanément.
Enregistrement numérique Enregistrer un nombre Enregistre des données numériques arbitraires.
Signet Enregistrer une URL Enregistre l'URL d'une page web en tant que signet.
Journal de temps Démarrer le suivi du temps Démarre une nouvelle session de suivi du temps.
Tâche Ajouter une tâche Crée une nouvelle tâche.
Dépense Enregistrer un montant Enregistre le montant et la description d'une dépense.
Humeur Enregistrer l'humeur (0–10) Enregistre votre humeur actuelle sur une échelle de 0 à 10.
Téléverser Téléverser un fichier Téléverse et enregistre des fichiers tels que des images ou des documents.
Conseil : Appuyez longuement sur le bouton "+" pour ouvrir directement le Bloc-notes.
Raccourci : Appuyez sur Entrée pour ouvrir la boîte de dialogue du Bloc-notes.
Conseil : Vous pouvez configurer les conditions de recherche pour la Vue d'agrégation et les tâches depuis le bouton "Tableau de bord" dans Paramètres.
