Page partagée

Cette page permet de consulter les enregistrements qu'un utilisateur de gkill a publiés via un lien de partage. Vous pouvez y accéder sans compte gkill.

À propos de cette page

  • Toute personne disposant du lien de partage peut consulter cette page.
  • Aucune connexion à gkill n'est requise.
  • Vous pouvez uniquement consulter les enregistrements partagés ; la modification et la suppression ne sont pas possibles.

Données affichées

Selon les paramètres du lien de partage, les enregistrements sont affichés dans l'une des vues suivantes.

Type de vue Description
Vue journal de vie Consulter les enregistrements partagés par ordre chronologique.
Vue des tâches Consulter les tâches partagées sous forme de tableau.
Conseil : L'étendue des enregistrements affichés est déterminée par les paramètres de l'utilisateur qui a créé le lien de partage.

Dépannage

Symptôme Solution
La page ne se charge pas Vérifiez que le lien de partage est correct. Le lien a peut-être expiré.
Aucune donnée n'est affichée Les paramètres de partage ont peut-être été modifiés. Veuillez vérifier auprès de la personne qui vous a envoyé le lien.
Un message d'erreur s'affiche Le serveur gkill est peut-être inaccessible. Veuillez patienter un instant et réessayer.
