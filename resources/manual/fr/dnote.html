Vue d'agrégation

Écran d'agrégation et de visualisation des données du journal de vie selon des conditions. Deux types peuvent être définis : les « Éléments » pour des valeurs agrégées individuelles et les « Listes » pour des agrégations groupées par clé.

Gestion des définitions

Élément Description
Onglets Chaque définition d'agrégation est affichée sous forme d'onglet. Cliquez sur un onglet pour basculer entre les définitions.
Nom de la définition Champ de texte permettant de modifier le nom de la définition d'agrégation.
Bouton ＋ (à côté des onglets) Ajoute une nouvelle définition d'agrégation.
Bouton supprimer Supprime la définition d'agrégation actuellement affichée.

Éléments de l'écran

Élément Description
Plage de dates Affiche la période utilisée pour l'agrégation.
Bouton de téléchargement Télécharge les résultats d'agrégation au format JSON.
Bouton ＋ (flottant) Affiche un menu pour choisir entre l'ajout d'un Élément ou d'une Liste.

Éléments (valeur agrégée unique)

Affiche les valeurs agrégées des données correspondant aux conditions au format : « Titre : Préfixe [Résultat agrégé] Suffixe ».

Paramètre Type Description
Titre Saisie de texte Définit le nom d'affichage de l'élément.
Préfixe Saisie de texte Texte affiché avant la valeur agrégée.
Suffixe Saisie de texte Texte affiché après la valeur agrégée.
Cible d'agrégation Liste déroulante Sélectionne la méthode d'agrégation (comptage, somme, moyenne, max, min, etc.).
Groupe de conditions Éditeur de conditions Filtre les cibles d'agrégation à l'aide de conditions AND/OR.

Listes (agrégation groupée)

Affiche les résultats d'agrégation groupés par clé sous forme de liste.

Paramètre Type Description
Titre / Préfixe / Suffixe Saisie de texte Mêmes paramètres que pour les Éléments.
Cible d'agrégation Liste déroulante Sélectionne la méthode d'agrégation.
Clé de groupement Liste déroulante Sélectionne le critère de groupement (titre, type de données, étiquette, mois, jour de la semaine, date, etc.).
Groupe de conditions Éditeur de conditions Filtre les cibles d'agrégation à l'aide de conditions AND/OR.

Édition du groupe de conditions

Élément Description
Logique de combinaison Sélectionnez AND / OR / NOT pour définir comment les conditions sont combinées.
Ajouter une condition Ajoute une nouvelle condition au groupe.
Ajouter un groupe Ajoute un groupe de conditions imbriqué.
Supprimer le groupe Supprime ce groupe de conditions.
Chaque condition Définit le type et la valeur de la condition (filtrer par type de données, étiquette, titre, montant, valeur d'humeur, etc.).

Opérations

Opération Description
Double-clic Ouvre l'éditeur de l'élément ou de la liste.
Clic droit Affiche le menu contextuel (Modifier / Supprimer).
Glisser-déposer Modifie l'ordre d'affichage des éléments et des listes.

Boutons d'action inférieurs

Bouton Description
Appliquer Enregistre toutes les modifications des définitions.
Annuler Annule les modifications et ferme le dialogue.
Astuce : Les Éléments sont utiles pour des valeurs agrégées uniques (ex : total des dépenses du mois), tandis que les Listes sont utiles pour des agrégations par groupe (ex : dépenses par magasin).
Astuce : Combiner les conditions permet de créer des requêtes d'agrégation complexes.
Astuce : Utilisez le glisser-déposer pour réordonner librement l'affichage des éléments et des listes.
Astuce : Les résultats d'agrégation peuvent être téléchargés en JSON pour une analyse avec des outils externes.
