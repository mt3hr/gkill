Ecran de connexion

Ecran d'authentification pour se connecter a gkill. Saisissez votre identifiant et votre mot de passe pour acceder a l'application.

Elements de l'ecran

Element Description
Champ d'identifiant Saisissez l'identifiant de votre compte gkill.
Champ de mot de passe Saisissez le mot de passe du compte.
Bouton de connexion Execute la connexion avec les identifiants saisis.
Affichage de la version Affiche le numero de version actuel de gkill dans le coin superieur droit.

Procedure

  1. Saisissez l'identifiant de votre compte dans le champ correspondant.
  2. Saisissez le mot de passe dans le champ correspondant.
  3. Cliquez sur le bouton "Connexion" ou appuyez sur Entree pour vous authentifier.
  4. Apres une authentification reussie, vous serez redirige vers la page par defaut configuree dans les parametres.
Astuce : Lors du premier lancement, un ecran d'inscription de compte s'affichera automatiquement. Suivez les instructions a l'ecran pour creer votre compte utilisateur et definir le mot de passe administrateur.
Astuce : Vous pouvez egalement vous connecter en appuyant sur Entree, ce qui permet de soumettre directement apres la saisie du mot de passe.
Dépannage : Si vous ne pouvez pas vous connecter, vérifiez votre identifiant et mot de passe.
