Premiers pas

gkill est une application permettant d'enregistrer et de consulter des journaux de vie quotidiens (life logs).

Ce que vous pouvez faire avec gkill

Type d'enregistrement Description
Mémo Enregistre des notes textuelles
Horodatage Enregistre les heures de debut et de fin
Tâche Gere les taches sur un tableau Kanban
Humeur Enregistre votre humeur avec une valeur de 0 a 10
Dépense Enregistre les montants et les details des depenses
Signet Sauvegarde des URLs de pages web
Enregistrement numérique Enregistre des donnees numeriques arbitraires
Fichier Televerse et enregistre des images ou des documents

Premier enregistrement de compte

  1. Lors du premier lancement de gkill, l'ecran d'enregistrement de compte s'affiche automatiquement.
  2. Entrez un identifiant utilisateur (ce sera le nom de votre compte).
  3. Entrez un mot de passe, puis entrez-le a nouveau pour confirmer.
  4. Definissez un mot de passe administrateur (utilise pour la gestion du serveur).
  5. Appuyez sur le bouton "Enregistrer" pour creer le compte.
  6. Vous serez redirige vers l'ecran de connexion. Connectez-vous avec le compte que vous venez de creer.
Conseil : Le compte admin est cree automatiquement lors du premier demarrage du serveur. Le compte que vous creez ici est un compte utilisateur normal.

Apres la connexion, l'ecran par defaut (Vue du journal de vie par defaut) s'affiche. Configurez les parametres de base tels que la langue et le theme sombre dans l'ecran de configuration.

Votre premier enregistrement en 60 secondes

  1. Connectez-vous.
  2. Maintenez le bouton "+" en bas a droite de l'ecran.
  3. Selectionnez "Bloc-notes".
  4. Saisissez une ligne de texte et appuyez sur Entree.
  5. Ouvrez "Vue du journal de vie" depuis le menu de navigation pour voir votre enregistrement.
Astuce : C'est tout ce qu'il faut pour votre premier enregistrement ! Une fois la syntaxe du bloc-notes maitrisee, vous pourrez enregistrer des taches, des valeurs d'humeur et plus encore de la meme maniere.

Creez votre premier enregistrement

  1. Après la connexion, l'écran par défaut spécifié dans les paramètres s'affiche.
  2. Appuyez sur le bouton "+" en bas a droite de l'ecran.
  3. Selectionnez "Bloc-notes".
  4. Saisissez ce que vous souhaitez enregistrer dans la zone de texte (par exemple : J'ai mange du curry au dejeuner aujourd'hui).
  5. Appuyez sur le bouton "Enregistrer" pour creer l'enregistrement.
Astuce : Un appui long sur le bouton "+" ouvre directement le bloc-notes.

Glossaire

Terme Description
Enregistrement Terme general pour les enregistrements dans gkill. Il englobe tous les types d'enregistrement, y compris les memos, les horodatages, les taches, etc.
Syntaxe bloc-notes Syntaxe de saisie de texte utilisee dans le bloc-notes. En ajoutant des symboles au debut de chaque ligne, vous pouvez creer differents types d'enregistrements en une seule fois
Horodatage Fonction de suivi du temps. Enregistre les heures de debut et de fin des travaux ou activites
Saihate Écran permettant d'ajouter des enregistrements en toute sécurité même depuis l'écran de verrouillage. Fonctionne comme un hub d'ajout d'enregistrements
Syntaxe du bloc-notes Designe le format base sur le texte pour saisir des enregistrements
Vue du journal de vie Ecran permettant de lister et rechercher des enregistrements
Vue d'agrégation Fonction d'affichage des agregations et statistiques de donnees
Informations associées Fonction qui recherche et affiche automatiquement les informations associees
Depot Emplacement de stockage des enregistrements. Vous pouvez utiliser plusieurs depots pour differents usages
Profil Ensemble de parametres regroupes par appareil
