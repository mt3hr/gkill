Vue du journal de vie

Ecran pour parcourir les journaux de vie avec des miniatures. Prend en charge plusieurs modes d'affichage et fonctions de filtrage.

Boutons de la barre d'outils

Bouton Description
☰ Barre laterale Ouvre/ferme la barre laterale pour le filtrage et la navigation.
📄 Details Affiche/masque le panneau de details de l'enregistrement.
📊 Vue recapitulative Affiche l'ecran d'analyse statistique.
📅 Calendrier Affiche la vue calendrier.
🗺 Carte Affiche les enregistrements avec des informations GPS sur la carte.
⚙ Parametres Ouvre l'ecran des parametres de l'application.
❔ AideAffiche l'aide de l'écran actuel.

Barre latérale de filtrage

La barre latérale offre de puissantes options de filtrage :

  • Plage de dates : Définissez les dates de début et de fin pour restreindre la période.
  • Type d'enregistrement : Affichez uniquement des types spécifiques (mémos, signets, fichiers, etc.).
  • Étiquettes : Filtrez par une ou plusieurs étiquettes.
  • Dépôt : Filtrez par dépôt de stockage des données.
  • Recherche textuelle : Recherchez dans le contenu des enregistrements.

Fonctions principales

  • Miniatures multicolonnes : Affichage des miniatures sur 1 a 10 colonnes.
  • Fonction de filtrage : Filtre les enregistrements par type, etiquette, plage de dates, etc. depuis la barre laterale.
  • Panneau de details : La selection d'un enregistrement affiche les informations detaillees.
  • Analyse statistique : Consultez les donnees statistiques et les tendances des enregistrements.
  • Affichage sur carte : Consultez les enregistrements avec des informations de localisation sur la carte.
Astuce : Vous pouvez ajuster le nombre de colonnes de miniatures (1-10) dans les parametres pour s'adapter a differentes tailles d'ecran.
Astuce : La valeur par défaut de la période d'affichage peut être modifiée dans le paramètre « Jours d'affichage du journal de vie » de l'écran de configuration.

Utilisation des étiquettes

Ajouter une étiquette

  1. Maintenez appuye ou faites un clic droit sur un enregistrement pour ouvrir le menu contextuel.
  2. Selectionnez "Ajouter une etiquette".
  3. Tapez le nom de l'etiquette (les etiquettes existantes s'affichent comme suggestions).

Modifier ou supprimer une étiquette

  1. Maintenez appuye ou faites un clic droit sur l'etiquette a modifier.
  2. Selectionnez "Modifier" ou "Supprimer" dans le menu.

Gestion de la structure des étiquettes

Vous pouvez gerer les noms d'affichage et l'ordre des etiquettes dans Parametres > Structure des etiquettes.

Astuce : En ecrivant #nom_etiquette dans la syntaxe du bloc-notes, l'enregistrement sera automatiquement etiquete. Consultez la Reference de syntaxe du bloc-notes pour plus de details.

Détails du téléchargement de fichiers

  1. Appuyez sur le bouton "+" et selectionnez "Telecharger un fichier".
  2. Selectionnez le depot de destination (depot pour la gestion de fichiers).
  3. Confirmez et ajustez la date et l'heure associees.
  4. Selectionnez le ou les fichiers a telecharger (plusieurs fichiers possibles).

Formats pris en charge : Images (JPEG/PNG/GIF/WEBP), videos (MP4, etc.), PDF, ZIP et autres.

Les fichiers ZIP peuvent etre explores dans leur contenu (clic droit > Ouvrir le contenu ZIP).

Limite de taille : Selon la configuration du serveur (-1 pour illimite).

Astuce : Appuyez sur Ctrl+V pour enregistrer le contenu du presse-papiers directement en tant que fichier.

Utilisation de la carte GPS

Telecharger un enregistrement GPS

  1. Appuyez sur le bouton "+" et selectionnez "Telecharger un fichier".
  2. Selectionnez le depot destine aux enregistrements GPS.
  3. Selectionnez le fichier GPX.
  4. Activez le bouton de carte pour confirmer l'affichage.

Configurer la cle API Google Maps

Allez dans Parametres > Parametres d'affichage > Cle API Google Maps et entrez votre cle.

Astuce : Obtenez votre cle API dans Google Cloud Console en activant l'API Maps JavaScript.
