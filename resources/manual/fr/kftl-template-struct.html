Structure des modèles

Écran de gestion de la structure des modèles de texte utilisés dans le bloc-notes. Vous pouvez organiser les modèles dans des dossiers et créer/modifier des textes prédéfinis pour une saisie rapide.

Éléments de l'écran

Élément Description
Arborescence des modèles Affiche les modèles et dossiers en structure hiérarchique. Les dossiers peuvent être ouverts et fermés d'un clic pour classer et organiser les modèles.

Opérations

Opération Description
Double-clic Ouvre l'éditeur du modèle sélectionné.
Clic droit Affiche le menu contextuel (Modifier / Supprimer).

Boutons d'action

Bouton Description
Ajouter un modèle Ajoute un nouvel élément de modèle.
Ajouter un dossier Ajoute un dossier pour organiser les modèles.
Appliquer Enregistre les modifications apportées à la structure des modèles.
Annuler Annule les modifications et ferme le dialogue.

Édition d'un élément de modèle

Double-cliquez sur un modèle ou sélectionnez « Modifier » dans le menu contextuel pour ouvrir l'éditeur.

Paramètre Type Description
Nom du modèle Saisie de texte Définit le nom d'affichage du modèle.
Contenu du modèle Zone de texte Le contenu textuel qui sera inséré dans le bloc-notes. Peut inclure la syntaxe du bloc-notes.

Menu contextuel

Élément du menu Description
Modifier Ouvre l'éditeur de l'élément de modèle.
Supprimer Supprime l'élément de modèle. Un dialogue de confirmation sera affiché.
Astuce : Inclure la syntaxe du bloc-notes dans les modèles permet une saisie structurée rapide depuis l'écran du bloc-notes.
Astuce : Utiliser des dossiers pour organiser les modèles par catégorie facilite l'accès.
Astuce : Les modifications ne sont pas enregistrées tant que vous n'appuyez pas sur Appliquer.
