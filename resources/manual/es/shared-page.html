Página compartida

Esta página permite ver los registros que un usuario de gkill ha publicado a través de un enlace compartido. Se puede acceder sin una cuenta de gkill.

Acerca de esta página

  • Cualquier persona con el enlace compartido puede ver esta página.
  • No es necesario iniciar sesión en gkill.
  • Solo se pueden ver los registros compartidos; no es posible editarlos ni eliminarlos.

Datos mostrados

Según la configuración del enlace compartido, los registros se muestran en una de las siguientes vistas.

Tipo de vista Descripción
Vista de registro de vida Ver los registros compartidos en orden cronológico.
Vista de tareas Ver las tareas compartidas en formato de tablero.
Consejo: El alcance de los registros mostrados está determinado por la configuración del usuario que creó el enlace compartido.

Solución de problemas

Síntoma Solución
La página no se carga Verifique que el enlace compartido sea correcto. Es posible que el enlace haya expirado.
No se muestran datos Es posible que la configuración de uso compartido haya cambiado. Consulte con la persona que le envió el enlace.
Se muestra un mensaje de error Es posible que no se pueda acceder al servidor de gkill. Espere un momento e inténtelo de nuevo.
