Primeros Pasos

gkill es una aplicacion para registrar y consultar registros de vida diaria (life logs).

Lo que puedes hacer con gkill

Tipo de registro Descripcion
Nota Registra notas de texto
Marca de tiempo Registra tiempos de inicio y fin
Tarea Gestiona tareas en un tablero Kanban
Estado de ánimo Registra tu estado de ánimo con un valor de 0 a 10
Gasto Registra importes y detalles de gastos
Marcador Guarda URLs de paginas web
Registro numérico Registra datos numéricos arbitrarios
Archivo Sube y registra imagenes o documentos

Primer registro de cuenta

  1. Al iniciar gkill por primera vez, la pantalla de registro de cuenta aparecera automaticamente.
  2. Ingrese un ID de usuario (este sera el nombre de su cuenta).
  3. Ingrese una contrasena y vuelva a ingresarla para confirmar.
  4. Ingrese una contrasena de administrador y confirmela (establece la contrasena de la cuenta admin creada automaticamente, utilizada para la gestion del servidor).
  5. Presione el boton "Registrar" para crear la cuenta.
  6. Sera redirigido a la pantalla de inicio de sesion. Inicie sesion con la cuenta que acaba de crear.
Consejo: La cuenta admin se crea automaticamente cuando el servidor se inicia por primera vez. Esta pantalla establece la contrasena de admin y crea su cuenta de usuario normal al mismo tiempo.

Despues de iniciar sesion, se mostrara la pantalla predeterminada (Vista de registro de vida por defecto). Configure los ajustes basicos como el idioma y el tema oscuro en la pantalla de configuracion.

Tu primer registro en 60 segundos

  1. Inicia sesion.
  2. Toca el boton "+" en la esquina inferior derecha de la pantalla.
  3. Selecciona "Bloc de notas" en el menu.
  4. Escribe lo que quieras registrar en el area de texto (por ejemplo: Hoy almorce curry).
  5. Pulsa el boton "Guardar" para crear el registro.
  6. Abre "Vista de registros" desde el menu de navegacion para ver tu registro.
Consejo: ¡Eso es todo lo que necesitas para tu primer registro! Una vez que aprendas la sintaxis del bloc de notas, podras registrar tareas, valores de estado de animo y mas de la misma manera. Consulta el Tutorial para mas detalles.
Consejo: Tocar el boton "+" abre un menu; mantener pulsado abre directamente el bloc de notas.

Glosario

Termino Descripcion
Registro Termino general para los registros en gkill. Abarca todos los tipos de registro, incluyendo notas, marcas de tiempo, tareas y mas
Sintaxis del bloc de notas Sintaxis de entrada de texto utilizada en el bloc de notas. Al agregar simbolos al inicio de cada linea, puedes crear diferentes tipos de registros a la vez
Marca de tiempo Funcion de seguimiento de tiempo. Registra los tiempos de inicio y fin de trabajos o actividades
Saihate Pantalla para añadir registros de forma segura incluso desde la pantalla de bloqueo. Funciona como centro de adición de registros
Vista de registros Pantalla para listar y buscar registros
Vista de Agregación Funcion para mostrar agregaciones y estadisticas de datos
Información Relacionada Funcion que busca y muestra automaticamente informacion relacionada
Repositorio Ubicacion de almacenamiento de registros. En la interfaz se muestra como «Repositorio». Puedes usar multiples repositorios para diferentes propositos
Perfil Conjunto de configuraciones agrupadas por dispositivo
