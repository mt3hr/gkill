Primeros Pasos > Configuración

Ajustes de la aplicacion

Pantalla para gestionar la configuracion general de la aplicacion y la estructura de datos. Puede ajustar las preferencias de visualizacion, opciones de comportamiento e integraciones externas.

Botones de accion superiores

Boton Descripcion
Configuracion del servidor (solo administradores) Abre la pantalla de configuracion del servidor para puerto, TLS, restricciones de acceso, etc. Solo visible para cuentas de administrador.
Recargar Actualiza los datos desde el servidor. Mantenga pulsado para forzar una recarga completa.
Cerrar sesion Finaliza la sesion actual y cierra sesion.

Configuracion general

Ajuste Tipo Descripcion
Idioma Desplegable Selecciona el idioma de la interfaz: ja (japones), en (ingles), zh (chino), ko (coreano), es (espanol), fr (frances), de (aleman).
Tema oscuro Casilla de verificacion Activa o desactiva el modo oscuro. Adecuado para entornos con poca luz.
Recarga en caliente Casilla de verificacion Cuando esta activado, la vista del registro de vida se actualiza automaticamente al detectar cambios en los datos.
Mostrar etiquetas en la lista Casilla de verificacion Controla si se muestran las etiquetas en las tarjetas de registros en las vistas de lista.
Mostrar pie de pagina de comparticion Casilla de verificacion Controla si se muestran las opciones de compartir en el area del pie de pagina.

Configuracion de visualizacion

Ajuste Tipo Descripcion
Periodo de visualizacion del registro de vida Casilla + Numero Establece el numero de dias de registros a mostrar en la vista del registro de vida. -1 para ilimitado. Un periodo mas corto carga mas rapido.
Pagina predeterminada Desplegable Elige la pagina que se muestra despues de iniciar sesion.
Columnas de imagenes del registro de vida Numero (1-10) Establece el numero de columnas de la cuadricula de miniaturas en la vista del registro de vida. Ajuste segun el tamano de su pantalla.
Tablero de tareas predeterminado Desplegable + Boton agregar Establece el tablero predeterminado en la pantalla de tareas. Use el boton de agregar para crear nuevos tableros.

Integracion externa

Ajuste Tipo Descripcion
Bookmarklet Solo lectura URL del bookmarklet JavaScript para la barra de marcadores del navegador. Permite guardar marcadores en gkill con un clic desde cualquier pagina web.
Clave API de Google Map Entrada de texto Introduce la clave API de Google Map para habilitar las funciones de mapa GPS. La vista de mapa requiere esta clave para funcionar.

Gestion de estructura de datos

Estos botones abren editores dedicados para gestionar como se organizan sus datos.

Boton Descripcion
Estructura de etiquetas Gestiona la jerarquia de etiquetas y sus agrupaciones. Puede crear, renombrar, reordenar y organizar etiquetas en relaciones padre-hijo.
Estructura de Repositorio Gestiona las ubicaciones de almacenamiento de datos (Repositorio). Puede agregar, eliminar o configurar donde se almacenan los registros.
Estructura de perfiles Gestiona las configuraciones de dispositivos y perfiles. Permite editar ajustes por dispositivo.
Estructura de tipos de registro Gestiona que tipos de registro se almacenan en que repositorios.
Estructura de plantillas Gestiona las plantillas de texto del bloc de notas para entrada rapida. Puede crear y editar textos predefinidos.
Vista de resumen Gestiona la agregacion de datos y las configuraciones de vistas de resumen.
Informacion relacionada Gestiona las estructuras de informacion relacionada.

Botones de accion inferiores

Boton Descripcion
Aplicar Guarda todos los cambios.
Cancelar Descarta los cambios y cierra la pantalla.

Pie de pagina

La parte inferior de la pantalla muestra informacion de la aplicacion:

  • Numero de version — Enlace clickeable a la pagina del proyecto.
  • Marca de tiempo de compilacion — Fecha y hora en que se compilo la aplicacion.
  • Hash del commit — El commit de Git a partir del cual se creo la compilacion.
Consejo: Despues de cambiar los ajustes, pulse "Aplicar" para guardar. Pulse "Cancelar" para descartar todos los cambios.
Consejo: Para telefonos moviles, establezca las columnas de imagenes en 1-2. Para pantallas de escritorio, 3-5 columnas funciona bien.
Consejo: Para usar las funciones de mapa GPS, habilite la API Maps JavaScript en Google Cloud Console e introduzca la clave API en Integracion externa.
