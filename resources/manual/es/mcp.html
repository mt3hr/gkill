Integración con IA

Al conectar asistentes de IA como Claude con gkill, puede buscar, agregar y editar sus registros de vida mediante conversación natural.

Lo que puede hacer

AcciónEjemplo
Buscar registros"Muéstrame las notas del almuerzo de la semana pasada" / "Resume los gastos de este mes"
Agregar registros"Agrega una nota para hoy" / "Crea una tarea"
Actualizar / eliminar registros"Marca esta tarea como completada"
Obtener GPS / archivos"Muéstrame mi ruta de hoy"
Verificar configuración"Lista mis plantillas"

Tipos de integración

TipoDescripción
Solo lecturaLa IA solo puede ver y buscar registros. No puede agregar ni modificar.
Solo escrituraLa IA solo puede agregar, actualizar y eliminar registros.
Lectura y escrituraLa IA puede ver, agregar, actualizar y eliminar registros.

Configuración (Claude Desktop)

Agregue lo siguiente al archivo de configuración de Claude Desktop (claude_desktop_config.json).

Integración de solo lectura

{
  "mcpServers": {
    "gkill": {
      "command": "npx",
      "args": ["-y", "gkill-mcp-server"],
      "env": {
        "GKILL_BASE_URL": "http://localhost:9999",
        "GKILL_USER": "su-id-de-usuario",
        "GKILL_PASSWORD_SHA256": "hash-sha256-de-su-contraseña"
      }
    }
  }
}
Consejo: GKILL_PASSWORD_SHA256 es el hash SHA256 de su contraseña. Puede generarlo con herramientas disponibles en su sistema operativo.

Configuración (Claude Code)

npx @anthropic-ai/claude-code mcp add gkill \
  --command npx \
  --args "-y,gkill-mcp-server" \
  --env GKILL_BASE_URL=http://localhost:9999 \
  --env GKILL_USER=su-id-de-usuario \
  --env GKILL_PASSWORD_SHA256=su-hash

Acceso remoto

Haga que su servidor gkill sea accesible a través de Internet y configure GKILL_BASE_URL con su dirección pública. Se recomienda encarecidamente habilitar TLS (HTTPS) por seguridad.

Consejo: La configuración de TLS se puede realizar desde Configuración del servidor.
