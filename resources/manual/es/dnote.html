Vista de agregación

Pantalla para agregar y visualizar datos del registro de vida según condiciones. Se pueden definir dos tipos: «Elementos» para valores agregados individuales y «Listas» para agregaciones agrupadas por clave.

Consejo: La vista de agregación es una función de alternancia dentro de la vista de registro de vida. Haga clic en el icono de agregación en la barra de herramientas de la vista de registro de vida para mostrarla.

Gestión de definiciones

Elemento Descripción
Pestañas Cada definición de agregación se muestra como una pestaña. Haga clic en una pestaña para cambiar.
Nombre de definición Campo de texto para editar el nombre de la definición de agregación.
Botón ＋ (junto a pestañas) Añade una nueva definición de agregación.
Botón eliminar Elimina la definición de agregación que se muestra actualmente.

Elementos de la pantalla

Elemento Descripción
Rango de fechas Muestra el período objetivo de la agregación.
Botón de descarga Descarga los resultados de agregación en formato JSON.
Botón ＋ (flotante) Muestra un menú para seleccionar añadir un elemento o añadir una lista.

Elementos (valor agregado individual)

Muestra el valor agregado de los datos que coinciden con las condiciones en el formato «Título: Prefijo [Resultado] Sufijo».

Configuración Tipo Descripción
Título Entrada de texto Establece el nombre visible del elemento.
Prefijo Entrada de texto Texto que se muestra antes del valor agregado.
Sufijo Entrada de texto Texto que se muestra después del valor agregado.
Objetivo de agregación Desplegable Selecciona el método de agregación (cantidad, suma, promedio, máximo, mínimo, etc.).
Grupo de condiciones Editor de condiciones Filtra los objetivos de agregación con condiciones AND/OR.

Listas (agregación por grupo)

Muestra los resultados agregados agrupados por clave en formato de lista.

Configuración Tipo Descripción
Título / Prefijo / Sufijo Entrada de texto Misma configuración que los elementos.
Objetivo de agregación Desplegable Selecciona el método de agregación.
Clave de grupo Desplegable Selecciona el criterio de agrupación (título, tipo de datos, etiqueta, mes, día de la semana, fecha, etc.).
Grupo de condiciones Editor de condiciones Filtra los objetivos de agregación con condiciones AND/OR.

Edición de grupo de condiciones

Elemento Descripción
Lógica de unión Seleccione AND / OR / NOT para configurar cómo se combinan las condiciones.
Añadir condición Añade una nueva condición al grupo.
Añadir grupo Añade un grupo de condiciones anidado.
Eliminar grupo Elimina este grupo de condiciones.
Cada condición Configura el tipo y valor de la condición (filtrar por tipo de datos, etiqueta, título, importe, valor de ánimo, etc.).

Operaciones

Operación Descripción
Doble clic Abre la pantalla de edición de un elemento o lista.
Clic derecho Muestra el menú contextual (editar/eliminar).
Arrastrar y soltar Permite cambiar el orden de visualización de elementos y listas.

Botones de acción inferiores

Botón Descripción
Aplicar Guarda los cambios de todas las definiciones.
Cancelar Descarta los cambios y cierra el diálogo.
Consejo: Los elementos son útiles para valores agregados individuales (ej: gastos totales del mes), y las listas para agregaciones por grupo (ej: gastos por tienda).
Consejo: Combinando condiciones se pueden crear criterios de agregación complejos.
Consejo: Con arrastrar y soltar puede cambiar libremente el orden de visualización de elementos y listas.
Consejo: Los resultados de agregación se pueden descargar como JSON para analizarlos con herramientas externas.
