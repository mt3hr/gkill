Gestión de cuentas

Esta pantalla gestiona las cuentas de usuario. Solo los administradores pueden acceder.

Botón superior

Pulse el botón Añadir cuenta para crear una nueva cuenta.

Lista de cuentas

Las cuentas registradas se muestran en una lista. Cada fila de cuenta contiene los siguientes elementos:

ElementoDescripción
Casilla de activar/desactivarAlterna el estado de la cuenta entre activada y desactivada
ID de usuarioMuestra el ID de usuario de la cuenta
Botón de asignación de almacenamientoAsigna repositorios de datos al usuario
Botón de restablecimiento de contraseñaGenera un enlace de restablecimiento de contraseña

Botón inferior

Cerrar: Cierra la pantalla de gestión de cuentas.

Consejo: La cuenta admin se crea automaticamente cuando el servidor se inicia por primera vez. Las cuentas agregadas desde la pantalla de gestion de cuentas son usuarios normales (sin privilegios de administrador). Las cuentas se gestionan habilitandolas/deshabilitandolas en lugar de eliminarlas.
