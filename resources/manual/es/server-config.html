Configuración del servidor

Esta pantalla configura el comportamiento del servidor. Solo los administradores pueden acceder.

Botón superior

Pulse el botón Gestión de cuentas para abrir la pantalla de gestión de cuentas.

Botones inferiores

Aplicar: Guardar la configuración.
Cancelar: Descartar los cambios y cerrar la pantalla.

Los cambios requieren reiniciar el servidor para que surtan efecto.
