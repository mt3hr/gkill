Panel

Una vista completa para ver todos los registros de vida de la fecha seleccionada de un vistazo. La mitad superior muestra la Vista de agregación y el mapa GPS de ese día, y la parte inferior muestra las tareas.

Botones de barra

Botón Descripción
← (Día anterior) Muestra los datos del día anterior.
Etiqueta de fecha (ej: 2025/6/15 Dom) Haga clic para abrir el selector de calendario.
→ (Día siguiente) Muestra los datos del día siguiente.
❔ Ayuda Muestra la ayuda de la pantalla actual.
⚙ Configuración Abre la configuración de la aplicación.
Menú de navegación (clic en título) Permite navegar a cada pantalla.

Áreas de visualización

Área Descripción
Vista de agregación Muestra la Vista de agregación del día seleccionado en la mitad izquierda. Puede limitar las condiciones de búsqueda en Configuración.
Mapa GPS Muestra el registro GPS del día seleccionado como mapa en la mitad derecha.
Lista de tareas Muestra las tareas relacionadas con el día seleccionado en la parte inferior. Puede limitar las condiciones de búsqueda en Configuración.

Botón de acción flotante (+)

Toque el botón "+" en la parte inferior derecha para abrir el menú de adición rápida de registros.

Elemento de menú Contenido Descripción
Bloc de notas Entrada de texto Abre la pantalla para ingresar notas y varios registros usando la sintaxis del bloc de notas.
Bloc de notas de tiempo Texto + marca de tiempo Abre la pantalla para gestionar la entrada de notas y las marcas de tiempo simultáneamente.
Registro numérico Registrar un número Registra datos numéricos arbitrarios.
Marcador Registrar una URL Guarda una URL de página web como marcador.
Registro de tiempo Iniciar seguimiento de tiempo Inicia una nueva sesión de seguimiento de tiempo.
Tarea Agregar una tarea Crea una nueva tarea.
Gasto Registrar un monto Registra el monto y descripción del gasto.
Estado de ánimo Registrar humor (0–10) Registra tu estado de ánimo actual como un valor de 0 a 10.
Cargar Cargar un archivo Carga y registra archivos como imágenes o documentos.
Consejo: Mantenga presionado el botón "+" para abrir el Bloc de notas directamente.
Atajo: Presione Enter para abrir el cuadro de diálogo del Bloc de notas.
Consejo: Puede configurar las condiciones de búsqueda para Vista de agregación y tareas desde el botón "Panel" en Configuración.
