Vista de registro

Pantalla que muestra los detalles de un registro individual. Puede consultar la informacion detallada de varios tipos de registros.

Botones de la barra de herramientas

Boton Descripcion
⚙ Ajustes Abre la pantalla de ajustes de la aplicacion.

Tipos de registro mostrados

Tipo de registro Descripcion
Nota Muestra el contenido de notas de texto.
Marcadores Muestra las URLs y titulos guardados.
Tareas Muestra el estado, fecha limite e informacion del tablero de la tarea.
Estado de animo Muestra el valor de animo de 0 a 10.
Gastos Muestra el monto y el contenido del gasto.
Registro de tiempo Muestra la hora de inicio/fin y el tiempo transcurrido.
Archivo Muestra una vista previa del archivo subido.
Registro de commit Git Muestra la informacion de commits del repositorio Git.

Informacion comun mostrada

En todos los tipos de registro se muestra la siguiente informacion:

  • Etiquetas: Lista de etiquetas asignadas al registro
  • Tiempo relacionado: Informacion de fecha y hora relacionada con el registro
  • Nombre del repositorio: Nombre del repositorio donde se almacenan los datos

Menu contextual

Al hacer clic derecho (o mantener pulsado) en un registro, se muestra un menu contextual que permite editar, eliminar, compartir y otras operaciones.

Consejo: Puede agregar o editar etiquetas directamente desde la pantalla de detalles del registro.
