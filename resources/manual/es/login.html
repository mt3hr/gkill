Pantalla de inicio de sesion

Pantalla de autenticacion para iniciar sesion en gkill. Introduzca su ID de usuario y contrasena para acceder a la aplicacion.

Elementos de la pantalla

Elemento Descripcion
Campo de ID de usuario Introduzca el ID de usuario de su cuenta gkill.
Campo de contrasena Introduzca la contrasena de la cuenta.
Boton de inicio de sesion Ejecuta el inicio de sesion con las credenciales introducidas.
Indicador de version Muestra el numero de version actual de gkill en la esquina superior derecha.

Procedimiento

  1. Introduzca el ID de usuario de su cuenta en el campo correspondiente.
  2. Introduzca la contrasena en el campo correspondiente.
  3. Haga clic en el boton "Iniciar sesion" o pulse Enter para autenticarse.
  4. Tras una autenticacion exitosa, sera redirigido a la pagina predeterminada configurada en los ajustes.
Consejo: En la primera ejecucion, se mostrara automaticamente una pantalla de registro de cuenta. Siga las instrucciones en pantalla para crear su cuenta de usuario y establecer la contrasena de administrador.
Consejo: Tambien puede iniciar sesion pulsando Enter, lo que permite enviar directamente despues de introducir la contrasena.
Solución de problemas: Si no puede iniciar sesión, verifique su ID de usuario y contraseña.
