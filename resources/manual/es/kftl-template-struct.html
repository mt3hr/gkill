Estructura de plantillas

Pantalla para gestionar la estructura de las plantillas de texto utilizadas en el bloc de notas. Puede organizar las plantillas en carpetas y crear/editar textos predefinidos para entrada rápida.

Elementos de la pantalla

Elemento Descripción
Árbol de plantillas Muestra plantillas y carpetas en estructura jerárquica. Las carpetas se pueden abrir y cerrar con un clic para clasificar y organizar las plantillas.

Operaciones

Operación Descripción
Doble clic Abre la pantalla de edición de la plantilla seleccionada.
Clic derecho Muestra el menú contextual (editar/eliminar).

Botones de acción

Botón Descripción
Añadir plantilla Añade un nuevo elemento de plantilla.
Añadir carpeta Añade una carpeta para organizar las plantillas.
Aplicar Guarda los cambios en la estructura de plantillas.
Cancelar Descarta los cambios y cierra el diálogo.

Edición de elementos de plantilla

Al hacer doble clic en una plantilla o seleccionar «Editar» en el menú contextual, se muestra la pantalla de edición.

Configuración Tipo Descripción
Nombre de plantilla Entrada de texto Establece el nombre visible de la plantilla.
Contenido de plantilla Área de texto Contenido de texto que se insertará en el bloc de notas. Puede incluir sintaxis del bloc de notas.

Menú contextual

Elemento del menú Descripción
Editar Abre la pantalla de edición del elemento de plantilla.
Eliminar Elimina el elemento de plantilla. Se muestra un diálogo de confirmación.
Consejo: Incluir sintaxis del bloc de notas en las plantillas permite realizar entradas estructuradas rápidamente desde la pantalla del bloc de notas.
Consejo: Usar carpetas para organizar las plantillas por categoría facilita su búsqueda.
Consejo: Los cambios en la estructura de plantillas no se guardan hasta pulsar el botón «Aplicar».
