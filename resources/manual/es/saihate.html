Inicio

Pantalla para registrar de forma segura incluso desde la pantalla de bloqueo. Funciona como centro para añadir rápidamente todo tipo de registros.

Saihate es un centro ligero especializado en agregar nuevos registros. No incluye funciones de navegación ni edición, por lo que no hay riesgo de modificar registros existentes por accidente. Funciona dentro de una sesión autenticada, lo que significa que solo es accesible mientras se ha iniciado sesión.

Cuándo usar cada pantalla: Use Saihate para anotar rápidamente una nota o entrada de tiempo, la Vista de Registro de Vida para buscar y explorar sus registros, y el Bloc de Notas para registrar en lote mediante entrada de texto.

Botones de la barra de herramientas

Boton Descripcion
🔄 Recargar Recarga la pantalla. Mantenga pulsado para forzar la recarga.
Cerrar sesion Cierra la sesion de la cuenta actual y vuelve a la pantalla de inicio de sesion.

FAB (Boton de accion flotante) "+"

Al hacer clic en el boton "+", se despliega un menu para crear rapidamente registros de los siguientes 9 tipos:

Tipo Descripcion
MemoCrea una nota de texto.
Tiempo+MemoCrea una nota con marca de tiempo.
NumericoRegistra datos numericos.
MarcadorGuarda un marcador de URL.
Registro de tiempoInicia una sesion de seguimiento de tiempo.
TareaCrea una nueva tarea.
GastoRegistra informacion de gastos.
AnimoRegistra un valor de animo (0-10).
Subir archivoSube un archivo.
Consejo: Mantenga pulsado el boton "+" para abrir directamente la pantalla de entrada del bloc de notas, lo que permite una entrada masiva rapida.
Atajo: Presione Enter en esta pantalla para abrir el diálogo del bloc de notas. Es conveniente para agregar registros rápidamente.
