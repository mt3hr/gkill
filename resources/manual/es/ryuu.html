Información relacionada

Pantalla para definir condiciones que buscan y muestran automáticamente registros relacionados con una entrada del registro de vida. Se pueden configurar consultas por definición para visualizar las relaciones entre registros.

Consejo: La información relacionada es una subvista que aparece al seleccionar un registro en la vista de registro de vida. Se puede acceder desde la vista de registro de vida.

Gestión de definiciones

Elemento Descripción
Pestañas Cada definición se muestra como una pestaña. Haga clic en una pestaña para cambiar.
Nombre de definición Campo de texto para editar el nombre de la definición.
Botón ＋ (junto a pestañas) Añade una nueva definición.
Botón eliminar Elimina la definición que se muestra actualmente.

Lista de consultas

Cada definición muestra sus consultas en una lista. Se pueden añadir nuevas consultas con el botón ＋ flotante.

Configuración de consultas

Configuración Tipo Descripción
Título Entrada de texto Establece el nombre visible de la consulta.
Prefijo Entrada de texto Texto que se muestra antes del valor coincidente.
Sufijo Entrada de texto Texto que se muestra después del valor coincidente.
Grupo de condiciones Editor de condiciones Filtra los objetivos de coincidencia con condiciones AND/OR.
Tipo de coincidencia de tiempo relacionado Desplegable Selecciona el método para determinar la relación temporal.
Período de búsqueda (horas) Entrada numérica Especifica el rango de tiempo en horas para buscar registros relacionados.
Consulta de búsqueda personalizada Casilla de verificación＋botón Active esta opción para usar una consulta de búsqueda propia. El botón abre el editor de consultas de búsqueda.

Tipos de coincidencia de tiempo relacionado

Tipo Descripción
Cercano en el tiempo Coincide con el registro más cercano en tiempo, tanto antes como después del registro objetivo.
Cercano en el tiempo (antes) Coincide con el registro más cercano en un tiempo anterior al registro objetivo.
Cercano en el tiempo (después) Coincide con el registro más cercano en un tiempo posterior al registro objetivo.

Botones de edición de consulta

Botón Descripción
Restablecer Revierte el contenido editado al estado original.
Guardar Guarda los cambios de la consulta.

Botones de acción inferiores

Botón Descripción
Aplicar Guarda los cambios de todas las definiciones y consultas.
Cancelar Descarta los cambios y cierra el diálogo.
Consejo: La información relacionada se utiliza para mostrar automáticamente otros registros relacionados al visualizar registros en la vista de registro de vida.
Consejo: Acortar el período de búsqueda hace que solo coincidan los registros temporalmente más cercanos.
Consejo: Puede crear múltiples definiciones para definir diferentes tipos de relaciones.
Consejo: Usar consultas de búsqueda personalizadas permite configurar condiciones de búsqueda más avanzadas.
