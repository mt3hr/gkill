시작하기 > 로그인

로그인 화면

gkill에 로그인하기 위한 인증 화면입니다. 사용자 ID와 비밀번호를 입력하여 애플리케이션에 접근합니다.

화면 요소

요소 설명
사용자 ID 입력란 gkill 계정의 사용자 ID를 입력합니다.
비밀번호 입력란 계정의 비밀번호를 입력합니다.
로그인 버튼 입력한 인증 정보로 로그인을 실행합니다.
버전 표시 화면 우측 상단에 gkill의 현재 버전 번호가 표시됩니다.

조작 절차

  1. 사용자 ID 입력란에 계정의 사용자 ID를 입력합니다.
  2. 비밀번호 입력란에 비밀번호를 입력합니다.
  3. "로그인" 버튼을 클릭하거나 Enter 키를 눌러 인증을 실행합니다.
  4. 인증에 성공하면 설정에서 지정한 기본 화면으로 이동합니다.
팁: 처음 실행 시 계정 등록 화면이 자동으로 표시됩니다. 화면의 안내에 따라 사용자 계정을 만들고 관리자 비밀번호를 설정하세요.
팁: Enter 키로도 로그인할 수 있으므로, 비밀번호 입력 후 바로 제출할 수 있습니다.
힌트: 로그인 세션은 30일간 유효합니다. 이 기간이 지나면 다시 로그인해야 합니다.
문제 해결: 로그인할 수 없는 경우 사용자 ID와 비밀번호를 확인하세요.
