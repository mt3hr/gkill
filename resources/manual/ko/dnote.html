집계 뷰

라이프로그 데이터를 조건에 따라 집계·시각화하는 화면입니다. 단일 집계값을 표시하는 '항목'과 키별로 그룹화하여 집계하는 '목록' 두 종류를 정의할 수 있습니다.

힌트: 집계 뷰는 라이프로그 뷰 화면 내의 토글 기능입니다. 라이프로그 뷰 화면의 도구 모음에 있는 집계 아이콘을 클릭하면 표시됩니다.

정의 관리

요소 설명
탭 각 집계 정의가 탭으로 표시됩니다. 탭을 클릭하여 전환합니다.
정의 이름 텍스트 필드에서 집계 정의의 이름을 편집할 수 있습니다.
＋ 버튼(탭 옆) 새 집계 정의를 추가합니다.
삭제 버튼 현재 표시 중인 집계 정의를 삭제합니다.

화면 요소

요소 설명
날짜 범위 집계 대상 기간을 표시합니다.
다운로드 버튼 집계 결과를 JSON 형식으로 다운로드합니다.
＋ 버튼(플로팅) 메뉴가 표시되며 항목 추가 또는 목록 추가를 선택할 수 있습니다.

항목(단일 집계값)

조건에 일치하는 데이터의 집계값을 「제목: 접두사 [집계 결과] 접미사」 형식으로 표시합니다.

설정 항목 종류 설명
제목 텍스트 입력 항목의 표시 이름을 설정합니다.
접두사 텍스트 입력 집계값 앞에 표시할 텍스트입니다.
접미사 텍스트 입력 집계값 뒤에 표시할 텍스트입니다.
집계 대상 드롭다운 집계 방법을 선택합니다(건수, 합계, 평균, 최대, 최소 등).
조건 그룹 조건 에디터 AND/OR 조건으로 집계 대상을 필터링합니다.

목록(그룹 집계)

키별로 그룹화하여 집계한 결과를 목록 형식으로 표시합니다.

설정 항목 종류 설명
제목 / 접두사 / 접미사 텍스트 입력 항목과 동일한 설정입니다.
집계 대상 드롭다운 집계 방법을 선택합니다.
그룹 키 드롭다운 그룹화 기준을 선택합니다(제목, 데이터 타입, 태그, 월, 요일, 날짜 등).
조건 그룹 조건 에디터 AND/OR 조건으로 집계 대상을 필터링합니다.

조건 그룹 편집

요소 설명
결합 로직 AND / OR / NOT을 선택하여 조건의 결합 방법을 설정합니다.
조건 추가 그룹에 새 조건을 추가합니다.
그룹 추가 중첩된 조건 그룹을 추가합니다.
그룹 삭제 이 조건 그룹을 삭제합니다.
각 조건 조건 타입과 값을 설정합니다(데이터 타입, 태그, 제목, 금액, 기분값 등으로 필터링).

조작 방법

조작 설명
더블클릭 항목 또는 목록의 편집 화면을 엽니다.
우클릭 컨텍스트 메뉴를 표시합니다(편집·삭제).
드래그 앤 드롭 항목이나 목록의 표시 순서를 변경할 수 있습니다.

하단 액션 버튼

버튼 설명
적용 모든 정의의 변경 사항을 저장합니다.
취소 변경 사항을 취소하고 대화상자를 닫습니다.
팁: 항목은 단일 집계값(예: 이번 달 지출 합계), 목록은 그룹별 집계 일람(예: 가게별 지출)에 사용하면 편리합니다.
팁: 조건을 조합하여 복잡한 집계 조건을 만들 수 있습니다.
팁: 드래그 앤 드롭으로 항목이나 목록의 표시 순서를 자유롭게 변경할 수 있습니다.
팁: 집계 결과를 JSON으로 다운로드하여 외부 도구에서 분석할 수도 있습니다.
