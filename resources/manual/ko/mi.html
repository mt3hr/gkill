작업 보드

칸반 스타일의 작업 관리 화면입니다. 작업의 상태와 진행 상황을 시각적으로 관리할 수 있습니다.

툴바 버튼

버튼 설명
☰ 사이드바 필터링 및 탐색을 위한 사이드바를 열고 닫습니다.
📄 상세 보기 작업의 상세 패널을 표시/숨깁니다.
📅 캘린더 캘린더 보기를 표시/숨깁니다.
⚙ 설정 애플리케이션 설정 화면을 엽니다.
❔ 도움말현재 화면의 도움말을 표시합니다.

주요 기능

  • 보드 열: 작업이 상태별로 열에 표시되며, 드래그 앤 드롭으로 상태를 변경할 수 있습니다.
  • 사이드바 필터: 태그, 보드, 상태 등의 조건으로 작업을 필터링합니다.
  • 상세 패널: 작업을 선택하면 상세 정보와 편집 옵션이 표시됩니다.
  • 캘린더 보기: 캘린더 형식으로 작업을 탐색합니다.
팁: 기본으로 표시할 보드는 설정 화면의 "기본 작업 보드 이름"에서 변경할 수 있습니다.

FAB(플로팅 액션 버튼) "+"

"+" 버튼을 클릭하면 메뉴가 펼쳐지며, 다음 유형의 기록을 빠르게 생성할 수 있습니다:

유형설명
메모텍스트 메모를 작성합니다.
시간+메모타임스탬프가 포함된 메모를 작성합니다.
수치수치 데이터를 기록합니다.
북마크URL 북마크를 저장합니다.
타임로그시간 추적 세션을 시작합니다.
작업새 작업을 생성합니다.
지출지출 정보를 기록합니다.
기분기분값(0-10)을 기록합니다.
업로드파일을 업로드합니다.
팁: "+" 버튼을 길게 누르면 메모장 입력 화면이 직접 열려 빠른 일괄 입력이 가능합니다.
단축키: 이 화면에서 Enter 키를 누르면 메모장 대화상자가 열립니다. 빠르게 기록을 추가할 때 편리합니다.
