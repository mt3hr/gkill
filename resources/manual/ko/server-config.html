저장소 관리

저장소는 gkill이 기록 데이터를 저장하는 디렉토리나 장소입니다. 용도（텍스트 메모용, 파일용, GPS 로그용 등）와 단말기별로 여러 저장소를 설정할 수 있습니다.

저장소 설정은 계정 관리 화면의 "저장소 할당 버튼"을 통해 각 사용자별로 수행합니다.

저장소 추가

  1. 계정 관리 화면을 열고 대상 사용자의 "저장소 할당 버튼"을 누릅니다.
  2. "추가" 버튼을 눌러 새 저장소를 추가합니다.
  3. 다음 항목을 설정합니다:
    • 표시 이름: 라이프로그 뷰와 필터에 표시되는 이름
    • 유형: 저장할 데이터의 종류（텍스트 메모, 타임스탬프, 작업, 파일 관리, GPS 로그, Git 커밋 로그 등）
    • 경로: 데이터를 저장하는 디렉토리 경로（파일 관리의 경우 glob 형식도 사용 가능）
    • 쓰기 대상으로 사용: 새 기록을 추가할 때 이 저장소에 저장할지 여부
    • 활성화: 이 저장소를 검색 및 표시 대상으로 할지 여부
  4. "저장" 버튼을 눌러 설정을 적용합니다.
