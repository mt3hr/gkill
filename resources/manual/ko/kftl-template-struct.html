템플릿 구조

메모장 화면에서 사용하는 텍스트 템플릿의 구조를 관리하는 화면입니다. 폴더로 템플릿을 정리하고, 빠른 입력을 위한 정형 문구를 작성·편집할 수 있습니다.

화면 요소

요소 설명
템플릿 트리 템플릿과 폴더를 계층 구조로 표시합니다. 폴더는 클릭하여 열고 닫을 수 있으며, 템플릿을 분류·정리할 수 있습니다.

조작 방법

조작 설명
더블클릭 선택한 템플릿의 편집 화면을 엽니다.
우클릭 컨텍스트 메뉴를 표시합니다(편집·삭제).

액션 버튼

버튼 설명
템플릿 추가 새 템플릿 요소를 추가합니다.
폴더 추가 템플릿을 정리하기 위한 폴더를 추가합니다.
적용 템플릿 구조의 변경 사항을 저장합니다.
취소 변경 사항을 취소하고 대화상자를 닫습니다.

템플릿 요소 편집

템플릿을 더블클릭하거나 컨텍스트 메뉴에서 「편집」을 선택하면 편집 화면이 표시됩니다.

설정 항목 종류 설명
템플릿 이름 텍스트 입력 템플릿의 표시 이름을 설정합니다.
템플릿 내용 텍스트 영역 메모장에 삽입되는 텍스트 내용입니다. 메모장 구문을 포함할 수 있습니다.

컨텍스트 메뉴

메뉴 항목 설명
편집 템플릿 요소의 편집 화면을 엽니다.
삭제 템플릿 요소를 삭제합니다. 확인 대화상자가 표시됩니다.
팁: 템플릿에 메모장 구문을 포함하면 메모장 화면에서 빠르게 구조화된 기록을 입력할 수 있습니다.
팁: 폴더를 사용하여 템플릿을 카테고리별로 정리하면 찾기 쉬워집니다.
팁: 템플릿 구조의 변경 사항은 「적용」 버튼을 누를 때까지 저장되지 않습니다.
