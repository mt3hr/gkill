표시되는 기록 유형

기록 유형 설명
메모 텍스트 메모의 내용을 표시합니다.
북마크 저장된 URL과 제목을 표시합니다.
작업 작업의 상태, 마감일, 보드 정보를 표시합니다.

타임로그 시작/종료 시간과 경과 시간을 표시합니다.
파일 업로드된 파일의 미리보기를 표시합니다.
Git 커밋 로그 Git 저장소의 커밋 정보를 표시합니다.

공통 표시 정보

모든 기록 유형에서 다음 정보가 표시됩니다:

  • 태그: 기록에 부여된 태그 목록
  • 관련 시간: 기록과 관련된 날짜/시간 정보
  • 저장소 이름: 데이터가 저장된 저장소의 이름

컨텍스트 메뉴

기록을 우클릭(또는 길게 누르기)하면 컨텍스트 메뉴가 표시되어 편집, 삭제, 공유 등의 작업을 수행할 수 있습니다.

팁: 기록 상세 화면에서 직접 태그를 추가하거나 편집할 수 있습니다.
