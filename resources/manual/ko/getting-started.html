시작하기

gkill은 일상의 라이프로그(생활 기록)를 기록하고 열람하기 위한 애플리케이션입니다.

gkill로 할 수 있는 것

기록 유형 설명
메모 텍스트 메모를 기록합니다
타임스탬프 시작 및 종료 시간을 기록합니다
작업 칸반 보드로 할 일을 관리합니다
기분 기분을 0~10의 수치로 기록합니다
지출 지출 금액과 내용을 기록합니다
북마크 웹 페이지 URL을 저장합니다
수치 기록 임의의 수치 데이터를 기록합니다
파일 이미지나 문서를 업로드하여 기록합니다

첫 계정 등록

  1. gkill을 처음 실행하면 계정 등록 화면이 자동으로 표시됩니다.
  2. 사용자 ID를 입력합니다(이것이 계정 이름이 됩니다).
  3. 비밀번호를 입력하고, 확인을 위해 한 번 더 입력합니다.
  4. 관리자 비밀번호를 입력하고 확인을 위해 한 번 더 입력합니다(자동 생성된 admin 계정의 비밀번호이며, 서버 관리에 사용됩니다).
  5. "등록" 버튼을 눌러 계정을 생성합니다.
  6. 로그인 화면으로 리디렉션되며, 방금 생성한 계정으로 로그인합니다.
힌트: admin 계정은 서버 첫 실행 시 자동으로 생성됩니다. 이 화면에서는 admin 계정의 비밀번호 설정과 일반 사용자 계정 생성을 동시에 수행합니다.

로그인 후, 기본 화면(초기 설정에서는 라이프로그 뷰)이 표시됩니다. 설정 화면에서 언어 및 다크 테마 등 기본 설정을 구성하세요.

60초 만에 첫 번째 기록

  1. 로그인합니다.
  2. 화면 오른쪽 하단의 "+" 버튼을 탭합니다.
  3. "메모장"을 선택합니다.
  4. 텍스트 영역에 기록하고 싶은 내용을 입력합니다 (예: 오늘 점심은 카레를 먹었다).
  5. "저장" 버튼을 누르면 기록이 생성됩니다.
  6. 탐색 메뉴에서 "라이프로그 뷰"를 열면 입력한 기록이 표시됩니다.
팁: 이것만으로 첫 번째 기록이 완료됩니다! 메모장 구문을 익히면 작업, 기분 값 등도 같은 방법으로 기록할 수 있습니다. 자세한 사용법은 튜토리얼을 참조하세요.
팁: "+" 버튼을 탭하면 메뉴가 열리고, 길게 누르면 메모장이 바로 열립니다.

용어집

용어 설명
기록 gkill에서 기록의 총칭입니다. 메모, 타임스탬프, 작업 등 모든 기록 유형을 포괄합니다
메모장 구문 메모장에서 사용하는 텍스트 입력 구문입니다. 각 줄 앞에 기호를 붙여 다른 종류의 기록을 한 번에 만들 수 있습니다
타임스탬프 시간 추적 기능입니다. 작업이나 활동의 시작 및 종료 시간을 기록합니다
Saihate 잠금 화면에서도 안전하게 기록을 추가할 수 있는 화면입니다. 각종 기록의 추가 허브로 기능합니다
라이프로그 뷰 기록을 목록으로 표시하고 검색하는 화면입니다
집계 뷰 데이터의 집계 및 통계를 표시하는 기능입니다
관련 정보 관련 정보를 자동으로 검색하여 표시하는 기능입니다
저장소 기록의 저장 위치입니다. 용도에 따라 여러 저장소를 사용할 수 있습니다
프로필 기기별 설정을 모아놓은 것입니다
