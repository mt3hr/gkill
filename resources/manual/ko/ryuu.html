관련 정보

라이프로그 기록에 대해 관련된 다른 기록을 자동으로 검색·표시하기 위한 조건을 정의하는 화면입니다. 정의별로 쿼리를 설정하여 기록의 관련성을 시각화할 수 있습니다.

힌트: 관련 정보는 라이프로그 뷰 화면에서 기록을 선택했을 때 표시되는 서브 뷰입니다. 라이프로그 뷰 화면에서 이용할 수 있습니다.

정의 관리

요소 설명
탭 각 정의가 탭으로 표시됩니다. 탭을 클릭하여 전환합니다.
정의 이름 텍스트 필드에서 정의의 이름을 편집할 수 있습니다.
＋ 버튼(탭 옆) 새 정의를 추가합니다.
삭제 버튼 현재 표시 중인 정의를 삭제합니다.

쿼리 목록

각 정의에는 쿼리가 목록으로 표시됩니다. 플로팅 ＋ 버튼으로 새 쿼리를 추가할 수 있습니다.

쿼리 설정 항목

설정 항목 종류 설명
제목 텍스트 입력 쿼리의 표시 이름을 설정합니다.
접두사 텍스트 입력 매치된 값 앞에 표시할 텍스트입니다.
접미사 텍스트 입력 매치된 값 뒤에 표시할 텍스트입니다.
조건 그룹 조건 에디터 AND/OR 조건으로 매치 대상을 필터링합니다.
관련 시각 매치 타입 드롭다운 시각의 관련성 판정 방법을 선택합니다.
검색 기간(시간) 숫자 입력 관련 기록을 검색할 시간 범위를 시간 단위로 지정합니다.
사용자 지정 검색 쿼리 체크박스＋버튼 독자적인 검색 쿼리를 사용할 경우 활성화합니다. 버튼으로 검색 쿼리 에디터를 엽니다.

관련 시각 매치 타입

타입 설명
시간적으로 가까움 대상 기록에 대해 시각이 전후로 가장 가까운 기록을 매치합니다.
시간적으로 가까움 (이전) 대상 기록보다 이전 시각에서 가장 가까운 기록을 매치합니다.
시간적으로 가까움 (이후) 대상 기록보다 이후 시각에서 가장 가까운 기록을 매치합니다.

쿼리 편집 버튼

버튼 설명
초기화 편집 내용을 원래대로 되돌립니다.

하단 액션 버튼

버튼 설명
적용 모든 정의와 쿼리의 변경 사항을 저장합니다.
취소 변경 사항을 취소하고 대화상자를 닫습니다.
팁: 관련 정보는 라이프로그 뷰 화면에서 기록을 열람할 때 관련된 다른 기록을 자동으로 표시하는 데 사용됩니다.
팁: 검색 기간을 짧게 설정하면 시간적으로 더 가까운 기록만 매치됩니다.
팁: 여러 정의를 만들어 다양한 종류의 관련성을 정의할 수 있습니다.
팁: 사용자 지정 검색 쿼리를 사용하면 더 고급 검색 조건을 설정할 수 있습니다.
