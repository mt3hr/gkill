애플리케이션 설정

애플리케이션 전체 설정과 데이터 구조를 관리하는 화면입니다. 표시 환경, 동작 옵션, 외부 연동 등을 조정할 수 있습니다.

상단 작업 버튼

버튼 설명
서버 설정 (관리자 전용) 포트, TLS, 접근 제한 등 서버 구성 화면을 엽니다. 관리자 계정에만 표시됩니다.
다시 불러오기 서버에서 데이터를 다시 불러옵니다. 길게 누르면 강제 다시 불러오기를 실행합니다.
로그아웃 현재 세션을 종료하고 로그아웃합니다.

일반 설정

설정 항목 유형 설명
언어 드롭다운 인터페이스 언어를 선택합니다: ja(일본어), en(영어), zh(중국어), ko(한국어), es(스페인어), fr(프랑스어), de(독일어).
다크 테마 체크박스 다크 모드를 켜거나 끕니다. 어두운 환경에서 사용하기에 적합합니다.
핫 리로드 체크박스 활성화하면 데이터 변경이 감지될 때 라이프 로그 뷰가 자동으로 새로고침됩니다.
목록에 태그 표시 체크박스 기록 목록에서 태그를 표시할지 여부를 제어합니다.
공유 푸터 표시 체크박스 푸터 영역에 공유 옵션을 표시할지 여부를 제어합니다.

표시 설정

설정 항목 유형 설명
라이프 로그 뷰 표시 일수 체크박스 + 숫자 라이프 로그 뷰에서 표시할 기록 일수를 설정합니다. -1로 설정하면 무제한입니다. 짧은 기간은 더 빨리 로드됩니다.
기본 페이지 드롭다운 로그인 후 표시되는 초기 페이지를 선택합니다.
라이프 로그 뷰 이미지 열 수 숫자 (1-10) 라이프 로그 뷰의 썸네일 그리드 열 수를 설정합니다. 화면 크기에 맞게 조정하세요.
기본 작업 보드 드롭다운 + 추가 버튼 작업 화면에서 기본으로 표시되는 보드를 설정합니다. 추가 버튼으로 새 보드를 만들 수 있습니다.

외부 연동

설정 항목 유형 설명
북마클릿 주소 읽기 전용 브라우저 북마크 바에 등록하는 JavaScript 북마클릿 URL입니다. 모든 웹페이지에서 한 번의 클릭으로 북마크를 저장할 수 있습니다.
Google Maps API 키 텍스트 입력 GPS 지도 기능을 활성화하기 위한 Google Maps API 키를 입력합니다. 지도 뷰에는 이 키가 필요합니다.

데이터 구조 관리

이 버튼들은 데이터 구성 방식을 관리하는 전용 편집기를 엽니다.

버튼 설명
태그 구조 태그 계층 구조와 태그 그룹을 관리합니다. 태그를 생성, 이름 변경, 재정렬하고 부모-자식 관계를 구성할 수 있습니다.
저장소 구조 데이터 저장 위치와 저장소를 관리합니다. 기록이 저장되는 위치를 추가, 제거 또는 구성할 수 있습니다.
프로필 구조 장치 및 프로필 구성을 관리합니다. 장치별 설정을 편집할 수 있습니다.
기록 유형 구조 각 기록 유형이 어떤 저장소에 저장되는지 관리합니다.
템플릿 구조 빠른 입력을 위한 메모장 텍스트 템플릿을 관리합니다. 자주 사용하는 텍스트 항목을 생성하고 편집할 수 있습니다.
집계 뷰 데이터 집계 방법과 집계 뷰 구성을 관리합니다.
관련 정보 관련 정보 구조를 관리합니다.

푸터

화면 하단에 애플리케이션 정보가 표시됩니다:

  • 버전 번호 — 클릭하면 프로젝트 페이지로 이동합니다.
  • 빌드 타임스탬프 — 애플리케이션이 빌드된 날짜와 시간입니다.
  • 커밋 해시 — 빌드의 기반이 된 Git 커밋입니다.
팁: 설정을 변경한 후 "적용" 버튼을 눌러 저장하세요. "취소"를 누르면 모든 변경 사항이 취소됩니다.
팁: 스마트폰에서는 이미지 열 수를 1-2열로, PC에서는 3-5열로 설정하면 보기 좋습니다.
팁: GPS 지도 기능을 사용하려면 Google Cloud Console에서 Maps JavaScript API를 활성화하고 API 키를 입력하세요.
