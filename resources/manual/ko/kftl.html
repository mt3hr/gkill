메모장

메모장 구문을 사용하여 생활 이벤트를 기록하는 메모장 화면입니다. 메모장 구문은 다양한 기록을 일괄 입력할 수 있는 행 구조 텍스트 형식입니다.

⚙ 설정 애플리케이션 설정 화면을 엽니다.

화면 요소

요소 설명
텍스트 입력 영역 메모장 구문으로 기록 내용을 입력하는 영역입니다.
템플릿 버튼 미리 정의된 메모장 템플릿 텍스트를 삽입합니다.
저장 버튼 저장 버튼을 클릭하여 입력된 메모장 텍스트를 분석하고 저장합니다.

메모장 구문에 대하여

메모장 구문은 gkill 고유의 텍스트 형식으로, 하나의 텍스트에서 여러 기록을 일괄 생성할 수 있습니다. 각 행은 하나의 명령 또는 데이터 항목을 나타냅니다.

팁: 템플릿 버튼을 사용하면 메모장 구문을 외우지 않아도 자주 사용하는 입력 형식을 빠르게 삽입할 수 있습니다.
