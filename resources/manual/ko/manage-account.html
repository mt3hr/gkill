계정 관리

사용자 계정을 관리하는 화면입니다. 관리자만 접근할 수 있습니다.

상단 버튼

계정 추가 버튼을 누르면 새 계정을 만들 수 있습니다.

계정 목록

등록된 계정이 목록으로 표시됩니다. 각 계정 행에는 다음 항목이 포함됩니다:

항목설명
활성화/비활성화 체크박스계정의 활성화 및 비활성화를 전환합니다
사용자 ID계정의 사용자 ID를 표시합니다
저장소 할당 버튼사용자에게 데이터 리포지토리를 할당합니다
비밀번호 재설정 버튼비밀번호 재설정 링크를 생성합니다

하단 버튼

닫기: 계정 관리 화면을 닫습니다.

힌트: admin 계정은 서버 첫 실행 시 자동으로 생성됩니다. 계정 관리 화면에서 추가된 계정은 일반 사용자(관리자 권한 없음)입니다. 계정은 삭제가 아닌 활성화/비활성화로 관리합니다.

비밀번호 재설정 절차

사용자가 비밀번호를 잊어버린 경우, 관리자가 다음 절차로 비밀번호 재설정 링크를 발급합니다.

관리자의 작업

  1. 서버 설정 화면에서 "계정 관리"를 엽니다.
  2. 대상 사용자 행의 "비밀번호 재설정" 버튼을 누릅니다.
  3. 확인 대화상자에서 "확인"을 누르면 비밀번호 재설정 링크가 생성됩니다.
  4. 표시된 재설정 링크를 복사하여 대상 사용자에게 공유합니다（메시지나 이메일 등을 통해）.

사용자의 작업

  1. 관리자로부터 받은 재설정 링크를 브라우저에서 엽니다.
  2. 새 비밀번호를 입력하고, 확인을 위해 한 번 더 입력합니다.
  3. "설정" 버튼을 누르면 새 비밀번호가 설정됩니다.
  4. 로그인 화면으로 이동하면 새 비밀번호로 로그인합니다.
힌트: 재설정 링크는 1회 한정으로 사용 가능합니다. 링크를 사용하면 무효화됩니다. 링크가 만료된 경우 관리자에게 재발급을 요청하세요.

저장소 할당

각 사용자에게 데이터를 저장할 "저장소"를 할당합니다. "저장소 할당 버튼"을 누르면 해당 사용자의 저장소를 설정하는 대화상자가 열립니다. 자세한 내용은 서버 설정의 "저장소 관리"를 참조하세요.
