AI 연동

Claude 등의 AI 어시스턴트와 gkill을 연동하면 자연어 대화로 라이프로그를 검색·추가·편집할 수 있습니다.

할 수 있는 것

작업예시
기록 검색"지난주 점심 메모 보여줘" / "이번 달 지출 정리해줘"
기록 추가"오늘 메모 추가해줘" / "할 일 만들어줘"
기록 수정 / 삭제"이 할 일 완료로 표시해줘"
GPS / 파일 가져오기"오늘 이동 경로 보여줘"
설정 확인"템플릿 목록 보여줘"

연동 유형

유형설명
읽기 전용AI가 기록을 조회·검색만 할 수 있습니다. 추가·변경은 불가.
쓰기 전용AI가 기록을 추가·수정·삭제만 할 수 있습니다.
읽기 및 쓰기AI가 기록 조회·추가·수정·삭제를 모두 할 수 있습니다.

설정 방법 (Claude Desktop)

Claude Desktop 설정 파일(claude_desktop_config.json)에 다음을 추가합니다.

읽기 전용 연동

{
  "mcpServers": {
    "gkill": {
      "command": "npx",
      "args": ["-y", "gkill-mcp-server"],
      "env": {
        "GKILL_BASE_URL": "http://localhost:9999",
        "GKILL_USER": "사용자-ID",
        "GKILL_PASSWORD_SHA256": "비밀번호의-SHA256-해시"
      }
    }
  }
}
팁: GKILL_PASSWORD_SHA256은 비밀번호를 SHA256으로 해시한 값입니다. 운영 체제에서 제공하는 도구로 생성할 수 있습니다.

설정 방법 (Claude Code)

npx @anthropic-ai/claude-code mcp add gkill \
  --command npx \
  --args "-y,gkill-mcp-server" \
  --env GKILL_BASE_URL=http://localhost:9999 \
  --env GKILL_USER=사용자-ID \
  --env GKILL_PASSWORD_SHA256=해시값

원격 접속

gkill 서버를 인터넷에서 접속 가능하게 설정하고 GKILL_BASE_URL에 공개 주소를 지정합니다. 보안을 위해 TLS(HTTPS) 활성화를 권장합니다.

팁: TLS 설정은 서버 설정에서 할 수 있습니다.
