대시보드

선택한 날짜의 라이프 로그를 한 화면에서 한눈에 볼 수 있는 종합 뷰입니다. 상단에는 해당 날짜의 집계 뷰와 GPS 지도, 하단에는 작업 목록이 표시됩니다.

툴바 버튼

버튼 설명
← (전날) 전날의 데이터를 표시합니다.
날짜 레이블（예: 2025/6/15 일） 클릭하면 달력 피커가 열립니다.
→ (다음 날) 다음 날의 데이터를 표시합니다.
❔ 도움말 현재 화면의 도움말을 표시합니다.
⚙ 설정 애플리케이션 설정을 엽니다.
내비게이션 메뉴（타이틀 클릭） 각 화면으로 이동할 수 있습니다.

표시 영역

영역 설명
집계 뷰 선택한 날짜의 집계 뷰를 왼쪽 절반에 표시합니다. 설정에서 검색 조건을 좁힐 수 있습니다.
GPS 지도 선택한 날짜의 GPS 로그를 오른쪽 절반에 지도로 표시합니다.
작업 목록 선택한 날짜와 관련된 작업을 하단에 목록으로 표시합니다. 설정에서 검색 조건을 좁힐 수 있습니다.

플로팅 액션 버튼（+）

화면 오른쪽 하단의 "+" 버튼을 탭하면 각종 기록 추가 메뉴가 표시됩니다.

메뉴 항목 내용 설명
메모장 텍스트 입력 메모장 구문으로 메모와 각종 기록을 입력하는 화면을 엽니다.
타임스탬프 메모장 텍스트＋타임스탬프 메모 입력과 진행 중인 타임스탬프를 동시에 관리하는 화면을 엽니다.
숫자 기록 숫자를 기록 임의의 숫자 데이터를 기록합니다.
북마크 URL을 기록 웹 페이지 URL을 북마크로 저장합니다.
타임스탬프 시간 측정을 시작 새 타임스탬프 세션을 시작합니다.
작업 작업을 추가 새 작업을 만듭니다.
지출 금액을 기록 지출 금액과 내용을 기록합니다.
기분 기분값（0〜10）을 기록 현재 기분을 0〜10의 숫자로 기록합니다.
업로드 파일을 업로드 이미지나 문서 등의 파일을 업로드하여 기록합니다.
팁: "+" 버튼을 길게 누르면 메모장이 직접 열립니다.
단축키: Enter 키를 누르면 메모장 대화상자가 열립니다.
팁: 설정 화면의 "대시보드" 버튼에서 표시할 요약 노트와 작업의 검색 조건을 설정할 수 있습니다.
