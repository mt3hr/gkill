Getting Started > Account Management

Account Management

This screen manages user accounts. Only administrators can access it.

Top Button

Press the Add Account button to create a new account.

Account List

Registered accounts are displayed in a list. Each account row contains the following items:

ItemDescription
Enable/Disable checkboxToggle the account between enabled and disabled
User IDDisplays the account's user ID
Storage Allocation buttonAssign data repositories to the user
Password Reset buttonGenerate a password reset link

Bottom Button

Close: Close the account management screen.

Tip: The admin account is automatically created when the server first starts. Accounts added from the Account Management screen are regular users (without admin privileges). Accounts are managed by enabling/disabling rather than deleting.
