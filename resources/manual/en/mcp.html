AI Integration

By connecting AI assistants such as Claude to gkill, you can search, add, and edit your life log records through natural conversation.

What You Can Do

ActionExample
Search records"Show me last week's lunch notes" / "Summarize this month's expenses"
Add records"Add a memo for today" / "Create a task"
Update / delete records"Mark this task as done"
Retrieve GPS / files"Show my route for today"
Check settings"List my templates"

Integration Types

TypeDescription
Read-onlyAI can view and search records only. Adding or modifying records is not allowed.
Write-onlyAI can add, update, and delete records only.
Read & WriteAI can view, add, update, and delete records.

Setup (Claude Desktop)

Add the following to the Claude Desktop configuration file (claude_desktop_config.json).

Read-only integration

{
  "mcpServers": {
    "gkill": {
      "command": "npx",
      "args": ["-y", "gkill-mcp-server"],
      "env": {
        "GKILL_BASE_URL": "http://localhost:9999",
        "GKILL_USER": "your-user-id",
        "GKILL_PASSWORD_SHA256": "sha256-hash-of-your-password"
      }
    }
  }
}
Tip: GKILL_PASSWORD_SHA256 is the SHA256 hash of your password. You can generate it using tools available on your operating system or online SHA256 calculators.

Setup (Claude Code)

Run the following command in your terminal.

# Add read-only integration
npx @anthropic-ai/claude-code mcp add gkill \
  --command npx \
  --args "-y,gkill-mcp-server" \
  --env GKILL_BASE_URL=http://localhost:9999 \
  --env GKILL_USER=your-user-id \
  --env GKILL_PASSWORD_SHA256=your-hash

Remote Access (Accessing from Outside Your Local Network)

Make your gkill server accessible over the internet, then set GKILL_BASE_URL to its public address. Enabling TLS (HTTPS) is strongly recommended for security.

Tip: TLS settings can be configured from Server Settings.
