In Progress

The Active Sessions screen shows all currently running time entry sessions (Time Log). Use it to monitor what you are currently doing and to stop sessions when activities end.

Toolbar

IconAction
⚙ SettingsOpen the application settings screen.
❔ HelpDisplay help for the current screen.

Active Session List

Each active session is displayed as a card showing:

  • Session name: The title of the time entry session.
  • Start time: When the session began.
  • Elapsed time: How long the session has been running.
  • Tags: Any tags associated with the session.

Stop Button

Each active session card has a stop button. Pressing it ends the session by recording the current time as the end time. The session then becomes a completed Time Log record viewable in the Life Log View and Record views.

FAB Quick-Add Menu

The floating action button (FAB) at the bottom-right provides quick access to create new records of any type:

TypeDescription
MemoCreate a new text memo.
Time + MemoStart a new Time Log session and attach a memo.
NumericRecord a numeric counter value.
BookmarkSave a URL bookmark.
Time LogStart a new time entry session.
TaskCreate a new task on a board.
ExpenseLog a financial expense.
MoodRecord your current mood (0-10 scale).
UploadUpload a file or image.
Tip: You can have multiple time entry sessions running simultaneously. This is useful when activities overlap, such as working on a project while in a meeting.
Tip: Check the Active Sessions screen regularly to make sure you have not forgotten to stop a session. Forgotten sessions can lead to inaccurate time records.
Shortcut: Press Enter on this screen to open the Memo Pad dialog. This is convenient for quickly adding records.
