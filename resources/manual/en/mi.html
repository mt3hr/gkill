Task Board

The Tasks screen provides a Kanban-style task board for managing tasks. Tasks are organized into boards and displayed as cards that can be tracked, checked off, and completed.

Toolbar

IconAction
☰ Sidebar ToggleShow or hide the filter sidebar for narrowing down displayed tasks.
📄 Detail View ToggleShow or hide the detail panel that displays full information for the selected task.
📅 Calendar ToggleShow or hide the calendar view.
⚙ SettingsOpen the application settings screen.
❔ HelpDisplay help for the current screen.

Board Columns

Tasks are organized into named boards (columns). Each board represents a category or project. You can configure boards in the application settings.

  • Tasks appear as cards within their assigned board column.
  • Each card shows the task title, status, limit date, and tags.
  • The default board can be set in the application settings.

Task Status

StatusDescription
OpenThe task is active and not yet started or in progress.
CheckedThe task has been marked as in-progress or acknowledged.
CompletedThe task is finished and done.

Sidebar Filters

The sidebar provides filtering options to narrow down the tasks displayed:

  • Board filter: Select which boards to show.
  • Tag filter: Filter tasks by their assigned tags.
  • Status filter: Show only open, checked, or completed tasks.
  • Date range: Filter tasks by their creation or limit date.

Detail Panel

When enabled, the detail panel shows full information for the currently selected task, including:

  • Task title and full description
  • Board assignment
  • Status and limit date
  • All associated tags
  • Related timestamps (Time Log sessions)
  • Edit and management controls
Tip: Set your most frequently used board as the "Default Task Board Name" in Settings so it opens automatically when you visit the Tasks screen.
Tip: Use tags to cross-reference tasks with other record types. For example, tag a task and related memos with the same tag to easily find all related items.

FAB "+" Quick-Add Menu

The floating action button (FAB) with a "+" icon at the bottom-right opens a quick-add menu for creating new records:

Record TypeDescription
MemoCreate a new text memo. Opens the memo creation dialog.
Time + MemoStart a new time tracking session and simultaneously attach a memo to it.
NumericRecord a numeric counter value (e.g., counting events or items).
BookmarkSave a URL with an optional title and notes.
Time LogStart a new time tracking session.
TaskCreate a new task and assign it to a board.
ExpenseLog a financial expense with amount and description.
MoodRecord your current mood on a scale of 0 to 10.
UploadUpload a file or image from your device.
Tip: Long-press the "+" FAB button to directly open the Memo Pad for fast batch entry, skipping the quick-add menu entirely.
Shortcut: Press Enter on this screen to open the Memo Pad dialog. This is convenient for quickly adding records.
