Record View

The Record view displays individual record details. A record is the core entity in gkill, representing any recorded life event. Each record has a type, timestamp, tags, and type-specific data.

Record Details

Each record view shows:

  • Timestamp: When the event occurred.
  • Tags: Associated tags displayed as chips. Tags help with filtering and organization.
  • Related Timestamps: Any linked time tracking sessions (Time Log) associated with this record.
  • Storage Name: The data storage where this record is stored.
  • Type-specific content: The main data of the record (text, URL, amount, etc.).

Context Menu

Right-click (or long-press on mobile) on a record to access the context menu with the following operations:

  • Edit: Open the record for editing.
  • Delete: Remove the record (moves to trash).
  • Share: Share the record via a shareable link.
  • Add Tag / Edit Tag: Attach a new tag or edit existing tags on the record.
  • Add Text / Edit Text: Attach a supplementary text note or edit existing text on the record.
  • Add Notification / Edit Notification: Set a reminder notification for this record, or edit an existing notification.
  • History: View the edit history of the record.
  • Copy ID: Copy the record ID to the clipboard.
Tip: Tags are a powerful way to organize your records across types. You can filter by tags in the Life Log View and Tasks views to find related records quickly.
