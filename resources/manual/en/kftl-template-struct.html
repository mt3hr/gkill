Template Structure

A screen for managing the structure of text templates used in the memo pad. You can organize templates into folders and create/edit boilerplate text for quick input.

Screen Elements

Element Description
Template Tree Displays templates and folders in a hierarchical structure. Folders can be expanded/collapsed by clicking, allowing you to categorize and organize templates.

Operations

Operation Description
Double-click Opens the editing screen for the selected template.
Right-click Displays the context menu (Edit / Delete).

Action Buttons

Button Description
Add Template Adds a new template element.
Add Folder Adds a folder for organizing templates.
Apply Saves the changes to the template structure.
Cancel Discards changes and closes the dialog.

Template Element Editing

Double-click a template or select "Edit" from the context menu to open the editing screen.

Setting Type Description
Template Name Text Input Sets the display name of the template.
Template Content Text Area The text content that will be inserted into the memo pad. Can include Memo Pad syntax.

Context Menu

Menu Item Description
Edit Opens the editing screen for the template element.
Delete Deletes the template element. A confirmation dialog will be displayed.
Tip: Include Memo Pad syntax in templates for quick structured input from the memo pad screen.
Tip: Use folders to organize templates by category for easier access.
Tip: Changes to the template structure are not saved until the Apply button is pressed.
