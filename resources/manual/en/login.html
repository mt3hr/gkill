Getting Started > Login

Login Screen

The login screen is the entry point for gkill. You must authenticate with a valid user account before accessing any features.

Screen Elements

ElementDescription
User IDEnter your registered user ID. This is case-sensitive.
PasswordEnter your password. The password is hashed (SHA-256) before being sent to the server.
Login ButtonSubmit your credentials to authenticate. On success, you will be redirected to your default page.
Version DisplayShows the current gkill version number and build information at the bottom of the screen.

Login Flow

  1. Enter your User ID in the first field.
  2. Enter your Password in the second field.
  3. Click the Login button or press Enter.
  4. If authentication succeeds, you will be taken to your configured default page (typically Home Hub).
  5. If authentication fails, an error message will be displayed.
First-time setup: On the very first run of gkill, an account registration screen will be displayed automatically. Follow the on-screen instructions to create your user account and set the administrator password.

Session Management

After a successful login, a session is created that lasts for 30 days by default. You do not need to log in again until the session expires or you explicitly log out.

Tip: Your session remains valid for 30 days after login. No re-login is needed until it expires.
Tip: If you are having trouble logging in, verify that the server is running and that you are connecting to the correct address and port (default: :9999).
Troubleshooting: If you cannot log in, verify your user ID and password.
