Getting Started

gkill is an application for recording and browsing daily life logs.

What You Can Do with gkill

Record Type Description
Memo Record text memos
Time Stamp Record start and end times
Task Manage tasks on a Kanban board
Mood Record your mood as a value from 0 to 10
Expense Record expense amounts and details
Bookmark Save web page URLs
Numeric Record Record arbitrary numeric data
File Upload and record images or documents

How to Launch

EnvironmentLaunch Method
WindowsRun gkill.exe or gkill_server.exe
Browser / PWAStart gkill_server and access http://localhost:9999 in your browser
AndroidInstall the APK and launch the app

First Account Registration

  1. When you launch gkill for the first time, the account registration screen will appear automatically.
  2. Enter a user ID (this will be your account name).
  3. Enter a password, then enter it again for confirmation.
  4. Enter an Admin Password and confirm it (this sets the password for the auto-created admin account, used for server management).
  5. Press the "Register" button to create your account.
  6. You will be redirected to the login screen. Log in with the account you just created.
Tip: The admin account is automatically created when the server starts for the first time. This screen sets the admin Admin Password and creates your regular user account at the same time.

After login, the default screen (Life Log View by default) will be displayed. Configure basic settings such as language and dark theme in the settings screen.

Your First Record in 60 Seconds

  1. Log in.
  2. Tap the "+" button at the bottom right of the screen.
  3. Select "Notepad" from the menu.
  4. Enter what you want to record in the text area (e.g., Had curry for lunch today).
  5. Press the "Save" button to create the record.
  6. Open "Life Log View" from the navigation menu to see your record.
Tip: That's all it takes for your first record! Once you learn the notepad syntax, you can record tasks, mood values, and more the same way. See the Tutorial for details.
Tip: Tapping the "+" button opens a menu; long-pressing opens the Notepad directly.

Glossary

Term Description
Record A general term for records in gkill. It encompasses all record types including memos, time stamps, tasks, and more
Notepad Syntax A text input syntax used in the Notepad. By adding symbols at the beginning of each line, you can create different types of records at once
Time Stamp A time tracking feature. Records the start and end times of work or activities
Saihate A screen for safely adding records even from the lock screen. Functions as a hub for adding various records
Life Log View A screen for listing and searching records
Aggregation View A feature for displaying data aggregation and statistics
Related Info A feature that automatically searches for and displays related information
Storage A storage location for records. You can use multiple storages for different purposes
Profile A collection of settings grouped by device
