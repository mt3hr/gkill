Time Tracking + Memo

This is a combined screen that merges the Memo Pad with the active sessions view. It is designed for efficient day-to-day logging: write Memo Pad text in the top half while monitoring your running time sessions in the bottom half.

Toolbar

IconAction
⚙ SettingsOpen the application settings screen.
❔ HelpDisplay help for the current screen.

Layout

SectionDescription
Top Half — Memo Pad InputA text input area for writing Memo Pad-formatted text. Works identically to the standalone Memo Pad screen. Includes a Template button and a Save button.
Bottom Half — Active SessionsDisplays currently running time tracking sessions (Time Log). Works identically to the Active Sessions screen. Each session shows its name, start time, elapsed duration, and a stop button.

Workflow

  1. Write Memo Pad text in the top area to record memos, start/stop timers, or create other record types.
  2. Click Save to submit the Memo Pad text.
  3. The active sessions list in the bottom half automatically refreshes after saving, showing any newly started or stopped sessions.
  4. Use the stop buttons in the bottom half to end specific sessions directly without writing Memo Pad text.

Auto-Refresh

After each save, the active sessions list refreshes automatically. This ensures the bottom half always reflects the latest state, including any sessions started or ended by the Memo Pad text you just submitted.

FAB "+" Quick-Add Menu

The floating action button (FAB) with a "+" icon at the bottom-right opens a quick-add menu for creating new records:

Record TypeDescription
MemoCreate a new text memo. Opens the memo creation dialog.
Time + MemoStart a new time tracking session and simultaneously attach a memo to it.
NumericRecord a numeric counter value (e.g., counting events or items).
BookmarkSave a URL with an optional title and notes.
Time LogStart a new time tracking session.
TaskCreate a new task and assign it to a board.
ExpenseLog a financial expense with amount and description.
MoodRecord your current mood on a scale of 0 to 10.
UploadUpload a file or image from your device.
Tip: Long-press the "+" FAB button to directly open the Memo Pad for fast batch entry, skipping the quick-add menu entirely.
Shortcut: Press Enter on this screen to open the Memo Pad dialog. This is convenient for quickly adding records.
Tip: Time Tracking + Memo is ideal as your daily driver screen. Set it as your default page in Settings so it opens automatically after login. You can quickly jot down memos and manage time tracking from a single view.
Tip: Use Memo Pad commands like start Meeting and end Meeting in the top area to control time sessions. They will immediately appear (or disappear) in the active sessions list below after saving.
