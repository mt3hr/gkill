Getting Started > Dashboard

Dashboard

A comprehensive view to see all your life logs for the selected date at a glance. The top half shows that day's Aggregation View and GPS map, and the bottom shows your tasks.

Toolbar Buttons

Button Description
← (Previous Day) Display data for the previous date.
Date label (e.g., 2025/6/15 Sun) Click to open the calendar picker.
→ (Next Day) Display data for the next date.
❔ Help Display help for the current screen.
⚙ Settings Open application settings.
Navigation menu (click title) Navigate to each screen.

Display Areas

Area Description
Aggregation View Displays the Aggregation View for the selected day in the left half. You can narrow search conditions in Settings.
GPS Map Displays the GPS log for the selected day as a map in the right half.
Task List Displays tasks related to the selected day in the bottom area. You can narrow search conditions in Settings.

Floating Action Button (+)

Tap the "+" button at the bottom right to open the quick-add menu for creating new records.

Menu Item Content Description
Notepad Text input Opens the screen for entering memos and various records using notepad syntax.
Time + Memo Text + time stamp Opens the screen for managing memo input and running time stamps simultaneously.
Numeric Record a number Records arbitrary numeric data.
Bookmark Record a URL Saves a web page URL as a bookmark.
Time Log Start time tracking Starts a new time tracking session.
Task Add a task Creates a new task.
Expense Record an amount Records an expense amount and description.
Mood Record mood (0–10) Records your current mood as a value from 0 to 10.
Upload Upload a file Uploads and records files such as images or documents.
Tip: Long-press the "+" button to open the Notepad directly.
Shortcut: Press Enter to open the Notepad dialog.
Tip: You can configure the search conditions for Aggregation View and tasks from the "Dashboard" button in Settings.
