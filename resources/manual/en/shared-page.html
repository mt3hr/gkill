Shared Page

This page is for viewing records that a gkill user has published through a shared link. You can access it without a gkill account.

About This Page

  • Anyone with the shared link can view this page.
  • No login to gkill is required.
  • You can only view shared records; editing or deleting is not possible.

Displayed Data

Depending on the shared link settings, records are displayed in one of the following views.

View Type Description
Lifelog View View shared records in chronological order.
Task View View shared tasks in a board format.
Tip: The range of displayed records is determined by the settings of the user who created the shared link.

Troubleshooting

Symptom Solution
Page does not load Check that the shared link is correct. The sharing settings may have been deleted. Please check with the person who sent you the link.
No data is displayed The sharing settings may have been changed. Please check with the person who sent you the link.
An error message is displayed The gkill server may be unreachable. Please wait a moment and try again.
