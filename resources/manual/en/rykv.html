Life Log View

The Life Log View is the primary life-log browsing screen. It displays all your recorded events as a timeline of image thumbnails and text entries, allowing you to review and explore your history.

Toolbar

IconAction
☰ SidebarShow or hide the filter sidebar for narrowing down displayed records.
📄 DetailShow or hide the detail panel for the currently selected record.
📊 Summary View StatisticsShow or hide the Summary View analytics panel with charts and statistics about your records.
📅 CalendarShow or hide the calendar heatmap for visual date-based navigation.
🗺 MapShow or hide the GPS map view to see records plotted on a geographic map.
⚙ SettingsOpen the application settings screen.

Thumbnail Grid

Records are displayed as a multi-column grid of thumbnails and text previews. You can configure the number of columns from 1 to 10 in the application settings.

ColumnsBest For
1-2Detailed reading on mobile devices. Shows more text per entry.
3-5Balanced view on tablets and small desktops.
6-10Overview browsing on large screens. Shows many records at once.

Filter Sidebar

The sidebar provides powerful filtering options:

  • Date range: Set start and end dates to narrow the time period.
  • Record type: Show only specific types (memos, bookmarks, files, etc.).
  • Tags: Filter by one or more tags.
  • Repository: Filter by data repository.
  • Text search: Search within record content.

Detail Panel

When enabled, selecting a record opens the detail panel showing its full content, tags, related timestamps, and edit controls. This is the same information shown in the Record view.

Summary View Analytics

The Summary View statistics panel shows charts and aggregate data about your records, including:

  • Record count trends over time
  • Type distribution breakdowns
  • Tag usage frequency
  • Activity patterns

Calendar Heatmap

The calendar view displays a heatmap where each day is colored according to the number of records. Click any date cell to jump to that day's records.

GPS Map View

If GPS data is available (from GPX logs or geotagged files), records are plotted on an interactive map. This requires a Google Maps API key configured in Settings.

Tip: The Life Log View display period can be configured in Settings. Set a shorter period for faster loading, or a longer period for broader browsing.
Tip: Adjust the number of image columns in Settings to match your screen size and browsing style. Fewer columns show more detail; more columns provide a broader overview.
Shortcut: Press Enter on this screen to open the Memo Pad dialog. This is convenient for quickly adding records.
