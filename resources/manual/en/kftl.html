Memo Pad

The Memo Pad uses a powerful text-based format for quickly recording multiple life events at once. This screen provides a text input area where you write formatted text and submit it to create records in batch.

Toolbar

IconAction
⚙ SettingsOpen the application settings screen.

Screen Elements

ElementDescription
Text Input AreaThe main text area where you write Memo Pad-formatted text. Each line is a statement.
Template ButtonInsert a predefined Memo Pad template into the text area. Templates are configured in Settings.
Save ButtonClick the Save button to submit the Memo Pad text to the server. All valid statements are parsed and saved as records.

Memo Pad Syntax Overview

The Memo Pad uses a line-structured text format. Each line represents a statement that creates or modifies a record. Lines are processed in order from top to bottom.

Example

start Deep Work
Had a productive coding session
tag programming
tag work
+3600s
end Deep Work
nlog 500 Coffee break snack
lantana 8

This example starts a "Deep Work" timer, records a memo with two tags, advances 1 hour, ends the timer, logs an expense, and records a mood of 8.

Tip: Use the Template button to insert frequently used Memo Pad patterns. Templates can be configured in the application settings and are a great way to speed up daily logging.
Tip: Lines are processed sequentially. Tags and text modifiers always apply to the most recently created record above them.
