Server Settings

This screen configures server behavior. Only administrators can access it.

Top Button

Press the Account Management button to open the account management screen.

Settings

SettingTypeDescription
ProfileDropdown + Add/DeleteSelect the active profile, create a new one, or delete an existing one
Local Access OnlyCheckboxRestrict connections to local access only
Notification ServiceCheckboxEnable the push notification service
Enable TLSCheckboxEnable HTTPS
Generate Self-signed TLSButtonCreate self-signed TLS certificate files
AddressTextServer listen address (e.g., :9999)
TLS CERT FileTextPath to the TLS certificate file
TLS KEY FileTextPath to the TLS private key file
Open Directory CommandTextOS command to open a directory
Open File CommandTextOS command to open a file
Bookmark TimeoutNumber (min: 5)Timeout in seconds for bookmark fetching
Bookmark User AgentTextUser-Agent string for bookmark fetching
User Data DirectoryTextPath to the user data storage directory

Bottom Buttons

Apply: Save the settings.
Cancel: Discard changes and close the screen.

Changes require a server restart to take effect.
