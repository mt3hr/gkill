チュートリアル

gkillの使い方を基本編と応用編に分けて解説します。初めての方は基本編から順にお読みください。

このダイアログについて: このヘルプダイアログはマウスやタッチ操作を背面に透過します。ダイアログを表示したまま、背面の画面を操作して実際に試すことができます。
スマートフォンでの利用: gkillはPWA（Progressive Web App）として動作します。ブラウザのメニューから「ホーム画面に追加」（iOS Safari）または「アプリをインストール」（Android Chrome）を選択すると、アプリのようにホーム画面から直接起動できます。

用語について

gkillでは、データの種類を「記録タイプ」（メモ、打刻、タスクなど）、操作する画面を「画面名」（メモ帳、打刻メモ帳、タスクなど）と呼びます。たとえば「メモ」はテキストメモという記録の種類を指し、「メモ帳」はその記録を入力するための画面を指します。「+」ボタンのメニューに表示される「メモ帳」「打刻帳」などは、それぞれの記録を追加するための画面名です。

画面の切り替え方

画面上部のツールバーに表示されているページタイトル（例: 「ライフログビュー」「タスク」など）をタップすると、ドロップダウンメニューが開き、各画面に移動できます。

基本編

1. gkillにログインする

初回起動時

gkillを初めて起動すると、アカウント登録画面が自動的に表示されます。

  1. ユーザーIDを入力します（これがあなたのアカウント名になります）。
  2. パスワードを入力し、確認のためもう一度入力します。
  3. 管理アカウントパスワードを入力し、確認のためもう一度入力します（自動作成される admin アカウントのパスワードです。サーバ管理に使用します）。
  4. 「登録」ボタンを押すと、アカウントが作成されます。
  5. ログイン画面にリダイレクトされるので、作成したアカウントでログインします。

通常のログイン

  1. ユーザーID入力欄にアカウントのユーザーIDを入力します。
  2. パスワード入力欄にパスワードを入力します。
  3. 「ログイン」ボタンをクリックするか、Enterキーを押します。
ヒント: ログイン後のセッションは30日間有効です。期限が切れるまで再ログインは不要です。

2. 記録を追加する

gkillではさまざまな種類のライフログを記録できます。画面右下の「+」ボタンから記録を追加します。

記録タイプ用途入力例
メモテキストメモを残す「今日は天気が良かった」
打刻時間の記録（開始・終了）「会議 10:00〜11:30」
タスクやるべきことを管理する「レポートを書く」
支出お金の使い方を記録する「コンビニ: おにぎり 150円」
気分気分を0〜10の数値で記録「7」
数値記録数値データを記録する「体重: 65.5」
ブックマークURLを保存する「https://example.com」
ファイル画像やファイルをアップロード写真、PDF など

記録の追加手順

  1. 画面右下の「+」ボタンをタップします。
  2. 追加したい記録タイプを選択します。
  3. 入力フォームに内容を記入します。
  4. 「保存」ボタンを押して記録を保存します。
ヒント: メモ帳画面を使えば、複数の記録をテキストでまとめて一括入力できます。詳しくはこの後の「3. メモ帳で複数記録をまとめて入力する」をご覧ください。

3. メモ帳で複数記録をまとめて入力する

メモ帳画面では、テキストを書くだけで複数の記録を一度に作成できます。

基本的な使い方

  1. メモ帳画面を開きます（「+」ボタンのメニューから「メモ帳」を選択）。
  2. テキストエリアに記録したい内容を入力します。
  3. 「保存」ボタン（！を入力して送信でも可）を押します。

テンプレートを使った入力

テンプレートを使えば、定型文をワンタッチで入力できます。

  1. メモ帳画面でテンプレート一覧からテンプレートを選択します。
  2. テンプレートの内容がテキストエリアに挿入されます。
  3. 必要に応じて内容を編集し、保存します。
ヒント: テンプレートはよく使う記録パターンを事前に登録しておくと便利です。設定画面の「テンプレート構造」から管理できます。

4. 記録を確認・検索する

ライフログビュー画面で、過去の記録を一覧表示・検索できます。

基本操作

  1. 画面上部のナビゲーションからライフログビュー画面を開きます。
  2. 記録が日付順に一覧表示されます。
  3. 日付を変更して、特定の日の記録を確認できます。

検索とフィルタリング

  • タグで絞り込み: 特定のタグが付いた記録だけを表示できます。
  • 日付範囲: 開始日と終了日を指定して期間を絞り込めます。
  • テキスト検索: キーワードを含む記録を検索できます。
ヒント: 設定画面の「ライフログビュー表示日数」を短くすると、読み込みが速くなります。

5. タスクを管理する

タスク画面では、カンバンボード形式でタスクを管理できます。

タスクの作成

  1. タスク画面を開きます。
  2. 「+」ボタンからタスクを追加します。
  3. タイトル、ボード名、期限などを入力します。
  4. 保存すると、指定したボードにタスクが表示されます。

タスクの完了

  1. 完了したタスクをタップして詳細を開きます。
  2. チェックマークを付けてタスクを完了にします。
ヒント: ボードを複数作成して、「仕事」「プライベート」「買い物」などカテゴリ別に管理すると整理しやすくなります。

6. 設定を変更する

設定画面からアプリケーションの動作をカスタマイズできます。

設定画面の開き方

  1. 画面上部のツールバーにある歯車アイコンをタップします。
  2. 設定画面が開きます。

よく使う設定

設定項目説明
言語UIの表示言語を変更します。
ダークテーマ暗い配色に切り替えます。
起動時の画面ログイン後に最初に表示するページを選択します。
ライフログビュー画像列数画像サムネイルの列数を調整します。
ヒント: 設定を変更したら「適用」ボタンを押してください。「キャンセル」を押すと変更が破棄されます。

応用編

8. テンプレートを活用する

よく使うメモ帳構文のパターンをテンプレートとして保存しておくと、毎回入力する手間が省けます。

テンプレートの管理

  1. 設定画面を開きます。
  2. 「テンプレート構造」ボタンをクリックします。
  3. テンプレートの追加・編集・削除ができます。

テンプレートの作成例

例えば、毎日の日記テンプレートを作成する場合：

。日記
、
ーら

このテンプレートを使えば、メモ帳画面でワンタッチで日記の雛形が入力され、メモ内容と気分値を追記するだけで記録できます。

ヒント: フォルダを使ってテンプレートをカテゴリ別に整理すると見つけやすくなります。

9. 打刻で時間を記録する

画面名 用途
打刻メモ帳 タイムスタンプ付きでメモを入力する画面。ファイルのアップロードも可能。
実行中 現在アクティブな打刻セッションの一覧。打刻の終了操作ができます。
メモ帳 メモ帳構文を使って打刻の開始・終了をテキストで入力。他の記録と一括入力可能。

打刻機能を使えば、作業時間や活動時間を正確に記録できます。「打刻メモ帳」はメモ入力と進行中の打刻管理を同時に行う専用画面で、「メモ帳」画面では打刻プレフィックス（ーた/ーえ）を使って他の記録と一括入力できます。

打刻の基本操作

  1. 開始: メモ帳で ーた プレフィックスを使うか、「+」メニューから打刻を追加します。タイトルと開始時刻を指定します。
  2. 終了: メモ帳で ーえ プレフィックスを使うか、打刻の編集画面で終了時刻を入力します。タイトルが一致する打刻が終了されます。

実行中の打刻を管理する

打刻一覧画面では、現在実行中の打刻を確認できます。

  • 実行中の打刻が一覧表示されます。
  • タップして終了時刻を設定できます。
ヒント: メモ帳構文の ーいえ を使うと、一致する打刻がない場合でもエラーになりません。存在確認なしで安全に終了できます。

10. 集計ビューでデータを分析する

ライフログビュー画面内の集計ビューでは、記録したデータを集計表で可視化できます。

集計ビューの使い方

  1. ライフログビュー画面のツールバーにある集計アイコンをクリックして、集計ビューを表示します。
  2. 集計の種類や期間を選択します。
  3. データが集計表で表示されます。

集計の設定

設定画面の「集計ビュー」ボタンから、集計方法やビューの構成を管理できます。

ヒント: 気分値や支出データの推移を時系列で確認すると、生活パターンが見えてきます。

11. 外部連携を活用する

ブックマークレットでURLを記録

ブラウザのブックマークバーにブックマークレットを登録すると、閲覧中のWebページをワンクリックでgkillに保存できます。

  1. 設定画面の「ブックマークレットアドレス」に表示されているURLをコピーします。
  2. ブラウザのブックマークバーに新しいブックマークとして登録します。
  3. 保存したいWebページを開いているときに、登録したブックマークをクリックします。

GPS記録と地図表示

GPSログを記録している場合、地図上に移動履歴を表示できます。

  1. 設定画面で「GoogleMapAPIキー」を設定します。
  2. GPSデータが記録されていれば、ライフログビューで地図表示が利用できます。
ヒント: GoogleMapAPIキーは、Google Cloud ConsoleでMaps JavaScript APIを有効にして取得してください。

12. 記録を共有する

gkillでは、記録の共有リンクを生成して他の人と共有できます。

共有リンクの生成手順

  1. 共有したい記録の詳細画面またはコンテキストメニューを開きます。
  2. 「共有」を選択します。
  3. 共有URLが生成されます。用途に応じて以下のURLを使い分けます。
URL種別用途
ローカルURL同じ端末内でのアクセス（localhost）
LAN URL同じネットワーク内の他の端末からのアクセス
WAN URLインターネット経由での外部アクセス

共有データの制御

共有時に、どのデータを含めるかを選択できます。

オプション説明
タグ記録に付けられたタグを共有に含める
テキスト記録のテキスト内容を共有に含める
打刻打刻の時間情報を共有に含める
位置情報GPS位置情報を共有に含める
時間のみ記録の時間情報のみを共有に含める
ヒント: LAN URLやWAN URLを使用するには、サーバ設定でサーバのアドレスを設定する必要があります。

共有リンクを受け取った方へ

共有リンクを開くと、gkillのアカウントがなくても記録を閲覧できます。

  • 表示される記録の範囲は、リンクを作成した人の設定によって決まります。
  • 記録の編集や削除はできません（閲覧のみ）。
  • リンクが無効になっている場合は、送信者に確認してください。

13. データのバックアップと管理

サーバ設定（管理者向け）

管理者アカウントでログインすると、設定画面に「サーバ設定」ボタンが表示されます。

  • ポート番号: サーバがリッスンするポートを変更できます（デフォルト: 9999）。
  • TLS設定: HTTPS通信を有効にできます。
  • アクセス制限: ローカルアクセスのみに制限できます。

アカウント管理

管理者は「アカウント管理」画面で、ユーザーアカウントの追加・有効化・無効化ができます。

データのバックアップ

gkillのデータはすべてホームディレクトリ（デフォルト: ~/gkill）に保存されています。このディレクトリをバックアップすれば、すべてのデータを保全できます。

ヒント: 定期的にホームディレクトリをバックアップすることを推奨します。特に datas/ と configs/ ディレクトリが重要です。
