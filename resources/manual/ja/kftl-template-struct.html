テンプレート構造

メモ帳画面で使用するテキストテンプレートの構造を管理する画面です。テンプレートをフォルダで整理し、クイック入力用の定型文を作成・編集できます。

画面の要素

要素 説明
テンプレートツリー テンプレートとフォルダを階層構造で表示します。フォルダはクリックで開閉でき、テンプレートを分類・整理できます。

操作方法

操作 説明
ダブルクリック 選択したテンプレートの編集画面を開きます。
右クリック コンテキストメニューを表示します（編集・削除）。

アクションボタン

ボタン 説明
テンプレート追加 新しいテンプレート要素を追加します。
フォルダ追加 テンプレートを整理するためのフォルダを追加します。
適用 テンプレート構造の変更を保存します。
キャンセル 変更を破棄してダイアログを閉じます。

テンプレート要素の編集

テンプレートをダブルクリックまたはコンテキストメニューから「編集」を選択すると、編集画面が表示されます。

設定項目 種類 説明
テンプレート名 テキスト入力 テンプレートの表示名を設定します。
テンプレート内容 テキストエリア メモ帳に挿入されるテキスト内容です。メモ帳構文を含めることができます。

コンテキストメニュー

メニュー項目 説明
編集 テンプレート要素の編集画面を開きます。
削除 テンプレート要素を削除します。確認ダイアログが表示されます。
ヒント: テンプレートにメモ帳構文を含めると、メモ帳画面で素早く構造化された記録を入力できます。
ヒント: フォルダを使ってテンプレートをカテゴリ別に整理すると見つけやすくなります。
ヒント: テンプレート構造の変更は「適用」ボタンを押すまで保存されません。
