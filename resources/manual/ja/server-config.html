サーバ設定

サーバの動作に関する設定を行う画面です。管理者のみアクセスできます。

上部ボタン

アカウント管理ボタンを押すと、アカウント管理画面が開きます。

設定項目

設定名種類説明
プロファイルドロップダウン + 追加/削除有効なプロファイルを選択、新規作成、または削除します
ローカルアクセスのみ許可チェックボックスローカル接続のみに制限します
通知サービス有効チェックボックスプッシュ通知サービスを有効にします
TLS有効チェックボックスHTTPSを有効にします
オレオレTLSファイル生成ボタン自己署名TLS証明書ファイルを生成します
アドレステキストサーバの待ち受けアドレス（例: :9999）
TLS CERTファイルテキストTLS証明書ファイルのパス
TLS KEYファイルテキストTLS秘密鍵ファイルのパス
ディレクトリを開くコマンドテキストディレクトリを開くOSコマンド
ファイルを開くコマンドテキストファイルを開くOSコマンド
ブックマークタイムアウト数値（最小: 5）ブックマーク取得時のタイムアウト（秒）
ブックマークユーザエージェントテキストブックマーク取得時のUser-Agent文字列
ユーザデータディレクトリテキストユーザデータの保存先ディレクトリのパス
月間アップロードサイズ上限数値月あたりのアップロードサイズ制限（-1で無制限）

下部ボタン

適用: 設定を保存します。
キャンセル: 変更を破棄して画面を閉じます。

設定の変更を反映するには、サーバの再起動が必要です。
