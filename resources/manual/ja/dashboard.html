ダッシュボード

選択した日付のライフログを一画面で俯瞰できる総合ビューです。上半分にはその日の集計ビューとGPS地図、下半分にはタスク一覧が表示されます。

ツールバーボタン

ボタン 説明
← (前日) 前の日付のデータを表示します。
日付ラベル（例: 2025/6/15(日)） クリックするとカレンダーピッカーが開きます。
→ (翌日) 次の日付のデータを表示します。
❔ ヘルプ 現在の画面のヘルプを表示します。
⚙ 設定 アプリケーション設定を開きます。
ナビゲーションメニュー（タイトルクリック） 各画面へ移動できます。

表示エリア

エリア 説明
集計ビュー 選択日の集計ビューを左半分に表示します。設定で検索条件を絞り込めます。
GPS地図 選択日のGPSログを右半分に地図表示します。
タスク一覧 選択日に関連するタスクを下部に一覧表示します。設定で検索条件を絞り込めます。

フローティングアクションボタン（+）

画面右下の「+」ボタンをタップすると、各種記録の追加メニューが表示されます。

メニュー項目 内容 説明
メモ帳 テキスト入力 メモ帳構文でメモや各種記録を入力する画面を開きます。
打刻メモ帳 テキスト＋打刻 メモ入力と進行中の打刻を同時に管理する画面を開きます。
数値記録 数値を記録 任意の数値データを記録します。
ブックマーク URLを記録 WebページのURLをブックマークとして保存します。
打刻帳 時間計測を開始 新しい打刻セッションを開始します。
タスク タスクを追加 新しいタスクを作成します。
支出 金額を記録 支出金額と内容を記録します。
気分 気分値（0〜10）を記録 現在の気分を0〜10の数値で記録します。
アップロード ファイルをアップロード 画像やドキュメントなどのファイルをアップロードして記録します。
ヒント: 「+」ボタンを長押しすると、メモ帳が直接開きます。
ショートカット: Enterキーを押すとメモ帳ダイアログが開きます。
ヒント: 設定画面の「ダッシュボード」ボタンから、表示する集計ビューやタスクの検索条件を設定できます。
