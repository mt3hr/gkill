設定画面

アプリケーション全体の設定やデータ構造の管理を行う画面です。表示・動作・外部連携に関するさまざまなオプションを調整できます。

上部アクションボタン

ボタン 説明
サーバ設定（管理者のみ） サーバーの詳細設定（ポート、TLS、アクセス制限など）を開きます。管理者権限を持つユーザーのみ表示されます。
再読込 データをサーバーから再読み込みします。長押し: 強制再読み込みを実行します。
ログアウト 現在のセッションを終了してログアウトします。

一般設定

設定項目 種類 説明
言語 ドロップダウン UIの表示言語を選択します。日本語(ja)、英語(en)、中国語(zh)、韓国語(ko)、スペイン語(es)、フランス語(fr)、ドイツ語(de)に対応しています。
ダークテーマ チェックボックス ダークモードの有効/無効を切り替えます。暗い環境での使用に適しています。
ホットリロード チェックボックス ライフログビュー画面のデータを自動的に更新する機能の有効/無効を切り替えます。変更が検出されると自動的に画面が更新されます。
リスト内タグ表示 チェックボックス 一覧画面で各記録に付与されたタグを表示するかどうかを設定します。
シェアフッター表示 チェックボックス フッター部分に共有オプションを表示するかどうかを設定します。

表示設定

設定項目 種類 説明
ライフログビュー表示日数 チェックボックス + 数値 ライフログビュー画面で表示する日数を設定します。-1で無制限になります。短い期間を設定すると読み込みが速くなります。
起動時の画面 ドロップダウン ログイン後に最初に表示するページを選択します。
ライフログビュー画像列数 数値（1〜10） ライフログビュー画面の画像サムネイル列数を設定します。端末の画面サイズに合わせて調整してください。
タスクデフォルト板名 ドロップダウン + 追加ボタン タスク画面で最初に表示するボードを選択します。追加ボタンで新しいボードを作成できます。

外部連携

設定項目 種類 説明
ブックマークレットアドレス 読み取り専用 ブラウザのブックマークバーに登録するJavaScriptブックマークレットのURLです。ブラウザからワンクリックでブックマークを追加する際に使用します。
Google Maps APIキー テキスト入力 GPS地図機能で使用するGoogle Maps APIキーを設定します。地図表示にはこのキーが必要です。

データ構造管理

アプリケーションのデータ構造を管理するボタン群です。各ボタンをクリックすると専用の編集画面が開きます。

ボタン 説明
タグ構造 タグの階層構造やタグのグループ分けを管理します。タグの親子関係や並び順を編集できます。
記録保管場所構造 データの保存先となるリポジトリを管理します。リポジトリの追加・削除・設定変更ができます。
プロファイル構造 デバイスやプロファイルの設定を管理します。各端末ごとの構成を編集できます。
記録タイプ構造 各記録タイプがどのリポジトリに保存されるかを管理します。
テンプレート構造 メモ帳のテキストテンプレートを管理します。クイック入力用の定型文を作成・編集できます。
集計ビュー データの集計方法や集計ビューの設定を管理します。
関連情報 関連情報の構造を管理します。

下部アクションボタン

ボタン 説明
適用 すべての変更を保存します。
キャンセル 変更を破棄して画面を閉じます。

フッター

画面下部にはアプリケーションの情報が表示されます。

  • バージョン番号 — クリックするとプロジェクトページへのリンクが開きます。
  • ビルドタイムスタンプ — ビルドが実行された日時です。
  • コミットハッシュ — ビルド元のGitコミットを示します。
ヒント: 設定を変更した後は「適用」ボタンを押して保存してください。「キャンセル」を押すと変更は破棄されます。
ヒント: スマートフォンではライフログビュー画像列数を1〜2列、PCでは3〜5列に設定すると見やすくなります。
ヒント: GPS地図機能を使うには、Google Cloud ConsoleでMaps JavaScript APIを有効にし、APIキーを取得して設定してください。
