関連情報

ライフログの記録に対して、関連する他の記録を自動的に検索・表示するための条件を定義する画面です。定義ごとにクエリを設定し、記録の関連性を可視化できます。

ヒント: 関連情報はライフログビュー画面で記録を選択した際に表示されるサブビューです。ライフログビュー画面から利用できます。

定義の管理

要素 説明
タブ 各定義がタブとして表示されます。タブをクリックして切り替えます。
定義名 テキストフィールドで定義の名前を編集できます。
＋ボタン（タブ横） 新しい定義を追加します。
削除ボタン 現在表示中の定義を削除します。

クエリ一覧

各定義にはクエリが一覧で表示されます。フローティングの＋ボタンで新しいクエリを追加できます。

クエリの設定項目

設定項目 種類 説明
タイトル テキスト入力 クエリの表示名を設定します。
プレフィックス テキスト入力 マッチした値の前に表示するテキストです。
サフィックス テキスト入力 マッチした値の後に表示するテキストです。
条件グループ 条件エディタ AND/OR条件でマッチ対象を絞り込みます。
関連時刻マッチタイプ ドロップダウン 時刻の関連性の判定方法を選択します。
検索期間（時） 数値入力 関連する記録を検索する時間範囲を時間単位で指定します。
カスタム検索クエリ チェックボックス＋ボタン 独自の検索クエリを使用する場合に有効にします。ボタンで検索クエリエディタを開きます。

関連時刻マッチタイプ

タイプ 説明
前後の最近時刻 対象の記録に対して時刻が前後で最も近い記録をマッチします。
直前の最近時刻 対象の記録よりも前の時刻で最も近い記録をマッチします。
直後の最近時刻 対象の記録よりも後の時刻で最も近い記録をマッチします。

クエリ編集のボタン

ボタン 説明
リセット 編集内容を元に戻します。
保存 クエリの変更を保存します。

下部アクションボタン

ボタン 説明
適用 すべての定義とクエリの変更を保存します。
キャンセル 変更を破棄してダイアログを閉じます。
ヒント: 関連情報は、ライフログビュー画面で記録を閲覧する際に、関連する他の記録を自動的に表示するために使用されます。
ヒント: 検索期間を短くすると、より時間的に近い記録のみがマッチします。
ヒント: 複数の定義を作成して、異なる種類の関連性を定義できます。
ヒント: カスタム検索クエリを使用すると、より高度な検索条件を設定できます。
