はじめに

gkillは、日々のライフログ（生活記録）を記録・閲覧するためのアプリケーションです。

gkillでできること

記録タイプ 説明
メモ テキストメモを記録します
打刻 時間の開始・終了を記録します
タスク やるべきことをカンバンボードで管理します
気分 気分を0〜10の数値で記録します
支出 支出金額と内容を記録します
ブックマーク WebページのURLを保存します
数値記録 任意の数値データを記録します
ファイル 画像やドキュメントをアップロードして記録します

初回アカウント登録

  1. gkillを初めて起動すると、アカウント登録画面が自動的に表示されます。
  2. ユーザーIDを入力します（これがあなたのアカウント名になります）。
  3. パスワードを入力し、確認のためもう一度入力します。
  4. 管理アカウントパスワードを入力し、確認のためもう一度入力します（自動作成される admin アカウントのパスワードです。サーバ管理に使用します）。
  5. 「登録」ボタンを押すと、アカウントが作成されます。
  6. ログイン画面にリダイレクトされるので、作成したアカウントでログインします。
ヒント: admin アカウントはサーバ初回起動時に自動作成されます。この画面では、admin アカウントの管理アカウントパスワード設定と、自分が使う通常ユーザアカウントの作成を同時に行います。

ログイン後、デフォルト画面（初期設定ではライフログビュー）が表示されます。設定画面で言語やダークテーマなどの基本設定を行ってください。

60秒で最初の記録

  1. ログインします。
  2. 画面右下の「+」ボタンをタップします。
  3. メニューから「メモ帳」を選択します。
  4. テキストエリアに記録したい内容を入力します（例: 今日のランチはカレーだった）。
  5. 「保存」ボタンを押すと記録が作成されます。
  6. ナビゲーションメニューから「ライフログビュー」を開くと、入力した記録が表示されます。
ヒント: これだけで最初の記録は完了です！メモ帳構文を覚えると、タスクや気分値なども同じ要領で記録できます。詳しい使い方はチュートリアルをご覧ください。
ヒント: 「+」ボタンをタップするとメニューが開きます。長押しするとメモ帳が直接開きます。

用語集

用語 説明

打刻 時間計測機能です。作業や活動の開始・終了時刻を記録します
さいはて ロック画面からでも安全に記録を追加できる画面です。各種記録の追加ハブとして機能します
ライフログビュー 記録を一覧表示・検索する画面です
集計ビュー データの集計・統計を表示する機能です
関連情報 関連する記録を自動検索・表示する機能です
記録保管場所 記録の保存先です。用途に応じて複数の記録保管場所を使い分けられます
プロファイル デバイスごとの設定をまとめたものです
