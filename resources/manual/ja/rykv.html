ライフログビュー画面

ライフログ全体を画像サムネイル付きで一覧表示する画面です。複数のフィルター列を使って、さまざまな条件でデータを同時に閲覧できます。

ツールバーボタン

ボタン 説明
☰ サイドバー 検索・フィルター用サイドバーの表示/非表示を切り替えます。
📄 詳細表示 選択した記録の詳細パネルの表示/非表示を切り替えます。
📊 集計ビュー 統計・分析ビューの表示/非表示を切り替えます。
📅 カレンダー カレンダー表示の表示/非表示を切り替えます。
🗺 地図 GPSログの地図表示の表示/非表示を切り替えます。
⚙ 設定 アプリケーション設定画面を開きます。
❔ ヘルプ 現在の画面のヘルプを表示します。

主な機能

画像サムネイル付き一覧表示

記録を複数列のグリッドレイアウトで表示します。画像やファイルが含まれる記録はサムネイルが表示されます。列数は設定画面で1〜10列に変更可能です。

サイドバー（フィルター・検索）

サイドバーで以下の条件を指定して記録を絞り込めます。

  • テキスト検索（キーワード）
  • タグによるフィルター
  • 期間の指定
  • 記録タイプの選択
  • 記録保管場所の選択

フィルター列の追加

「+」ボタンで新しいフィルター列を追加できます。異なるフィルター条件の結果を横に並べて比較できます。

詳細パネル

記録を選択すると、詳細パネルにその記録の全情報が表示されます。

集計ビュー統計

記録データの統計・分析結果を表示します。記録の傾向やパターンを把握できます。

GPS地図表示

GPSログデータを地図上に表示します。移動経路や位置情報を視覚的に確認できます。Google Maps APIキーの設定が必要です。

ヒント: 画像列数は設定画面で変更できます。スマートフォンでは1〜2列、PCでは3〜5列がおすすめです。
ヒント: フィルター列を複数追加して、異なる条件の記録を並べて比較すると便利です。
ヒント: 表示期間のデフォルト値は設定画面の「ライフログビュー表示日数」で変更できます。

フローティングアクションボタン（+）

画面右下の「+」ボタンをタップすると、各種記録の追加メニューが表示されます。

タグの使い方

タグは記録を分類・整理するためのラベルです。複数のタグを1つの記録に付けられます。

タグを追加する

  1. タイムライン上の記録を長押し（モバイル）または右クリック（デスクトップ）します。
  2. コンテキストメニューから「タグを追加」を選択します。
  3. タグ名を入力して追加します。既存のタグは入力候補として表示されます。

タグを編集・削除する

  1. タイムライン上のタグを長押し（モバイル）または右クリック（デスクトップ）します。
  2. コンテキストメニューから「タグを編集」または「タグを削除」を選択します。

タグの階層構造

設定画面の「タグ構造」でタグの表示名や並び順を管理できます。よく使うタグを上位に置くと、タグの入力候補一覧で見つけやすくなります。

ヒント: メモ帳構文で #タグ名 と入力すると、テキスト送信時に自動でタグが付きます。詳しくはメモ帳構文リファレンスをご覧ください。

ファイルのアップロード詳細

「+」ボタンメニューの「アップロード」から、画像・動画・ドキュメントなどのファイルをgkillに保存できます。

操作手順

  1. 「+」ボタンから「アップロード」を選択します。
  2. アップロード先の記録保管場所（ファイル管理用のリポジトリ）を選択します。
  3. 関連日時（ファイルの記録日時）を確認・調整します。
  4. ファイルを選択してアップロードします。複数ファイルを一度に選択できます。

対応ファイル形式

画像（JPEG・PNG・GIF・WEBP）、動画（MP4等）、PDF、ZIP、およびその他の一般的なファイル形式に対応しています。ZIPファイルはアップロード後に内容をブラウズできます（記録を右クリック→「ZIP内容を開く」）。

アップロードサイズ制限

1か月あたりのアップロードサイズ上限はサーバ設定で管理者が設定します。上限に達した場合はエラーメッセージが表示されます（-1 に設定されている場合は無制限です）。

ショートカット: Ctrl+V を押すと、クリップボードにある画像やテキストをそのままファイルとして保存できます。スクリーンショットを素早く記録するのに便利です。

GPS地図の使い方

ツールバーの地図ボタンをオンにすると、記録の期間に対応するGPSの移動経路が地図上に表示されます。

GPSログのアップロード手順

  1. 「+」ボタンから「アップロード」を選択します。
  2. アップロード先に「GPS記録用」の記録保管場所（GPSログ用のリポジトリ）を選択します。
  3. GPXファイルを選択してアップロードします。
  4. アップロード後、地図ボタンをオンにすると移動経路が表示されます。

Google Maps APIキーの設定

地図表示にはGoogle Maps APIキーが必要です。設定画面（ライフログビュー画面のツールバー「設定」→「表示設定」）の「Google Maps APIキー」にAPIキーを入力して保存してください。

ヒント: Google Maps APIキーは Google Cloud Console で取得できます。Maps JavaScript API を有効にしたキーが必要です。
